View Settings - Bridge Model
Plan

Customize the plan view of the bridge model by enabling or disabling the various view attributes.

Label Piers

When checked, the name of each pier is shown

Label Girders

When checked, the label (A-Z) for each girder is shown 

Label Girder Bearings

When checked, the bearings (N 34 56 23 E) of each girder is shown 

North Up

When checked North is always drawn vertically, otherwise, the North is rotated to best fit the bridge into the window.
