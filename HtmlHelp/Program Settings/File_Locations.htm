Program Settings - File Locations

PGSuper uses several user defined files and folders. You can customize PGSuper by specifying these files and folders in the File Locations tab.

Templates Folder

Enter the root folder where shared project templates are stored. If you are sharing project templates with other engineers in your organization, your PGSuper Administrator can supply you with the location of the Workgroup Templates Folder. See examples above for valid folder names.

Examples

Folder in a local hard disk or mapped network drive

M:\MyPGSuperTemplates\

Network folder

\\myserver\MyPGSuperTemplates\

FTP Server

ftp://ftp.myserver.com/MyPGSuperTemplates/

Master Library File

Enter the fully qualified name of the PGSuper library file. If you are sharing a library with other engineers in your organization, your PGSuper Administrator can supply you with the name of the PGSuper library file.

Examples

Folder in a local hard disk or mapped network drive

M:\MyPGSuperTemplates\PGSuper.lbr

Network folder

\\myserver\MyPGSuperTemplates\PGSuper.lbr

FTP Server

ftp://ftp.myserver.com/MyPGSuperTemplates/PGSuper.lbr

Shared Libraries and Templates

PGSuper can use shared libraries and templates so all engineers working for or with a particular organization will have the same information. To configure PGSuper to use shared Templates and Master Libraries:

  1. Press the View catalog of Template and Library Publishers button.

  2. Select a publisher from the list and press the OK button

  3. The Workgroup Templates Folder and the Master Library File fields will be updated

  4. Select how often you would like PGSuper to check for updates

  5. Confirm that the Update templates and libraries immediately box is check

When you exit the Program Settings dialog, PGSuper will download the Templates and Master Library files.
