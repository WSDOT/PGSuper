Structural Analysis Models

This section describes the analytical models used by PGSuper to calculate structural analysis results.

Model Geometry

PGSuper uses a 2D line model for structural analysis. The length of the spans in the analysis model are equal to the Centerline Bearing to Centerline Bearing length in the bridge model. The centerlines of bearing are located using the information in the Connection library entries used in the bridge model.

Span Length Mapping

The 2D line model uses the same girder in each span (e.g. Girder B from every span is used to create the 2D line model for the first interior girder line). When the number of girders per span is not equal the right exterior girder is used in the 2D line model when a span has fewer girders then the girder line for which the model is being constructed. Bridge Site Stage 1

The concentrated loads P and M are the loads induced by the cantilever portion of the girder and the end diaphragms. Because PGSuper does not impose a restriction on the length of the overhangs or the size of the girders or diaphragms, the induced moments and reactions can be significant, therefore they will be modeled.

Material properties for concrete are based on the 28 day strength. It is assumed that all losses have occurred by this stage.

Slab Load in Main Span

Loads from the main slab on interior girders are uniform along the entire length of the girder.

However, if the bridge is curved, the tributary width of the slab can change along the length of the girder as the edge of deck curves towards or away from the exterior girder. Loads from the main slab for exterior girders are approximated with segments of uniform loads applied along the span. The load value for each segment is equal to the average tributary width taken at the ends of the segment.

For both types of girders, the main slab load is approximated as segments of uniform loads applied along the span.

 

Slab Pad Load in Main Span

The slab pad, or haunch load accounts for the mass of concrete between the top flange of the girder and the bottom of the main slab. For purposes of determining this load, it is assumed that the top of girder is flat (e.g. there is no camber). The depth of the haunch is varied for roadway geometry effects only. The depth of the slab pad is taken to be the difference in elevation between the roadway surface and the top of girder (assuming no camber) less the gross depth of the slab. The load is then computed as the top flange width times the depth of the slab pad times the unit weight of the slab concrete.

Shear Key Loads

Shear key loads can be automatically generated only for adjacently-spaced TxDOT Box girders. To activate the load, specify the ShearKeyDepth variable in the Box Girder library entry.

  • The shear key will fill the gap between girders from the specified shear key depth to the tops of the beams and will have the same unit weight as the slab concrete.

  • The load is applied in a load case called "Shear Key" applied in Bridge Site stage 1, and added to the DC combination.

  • Structural stiffness properties of the shear key are not considered in any analyses.

  • The width of the shear key will be variable if the joint spacing between girders is variable.

  • The shear key load is applied between bearings. No cantilever portion is applied.

  •  Results will only be displayed if a shear key is present

Bridge Site Stage 2

Superimposed dead loads are applied to the bridge in the stage 2 model. The superimposed dead loads consist of the railing system and the overlay.

Railing System Loads

Railing system loads consist of dead loads from the traffic barrier, sidewalk, and combination (interior) railing as well as pedestrian live load on sidewalk. These loads are distributed to a maximum of n "nearest" girders, mating surfaces, or webs of the bridge section.

A mating surface is the contact point between a girder and the bridge deck. Examples are:

Exterior and Interior Traffic Barriers

Dead loads of exterior and interior barriers are distributed as follows:  

Distribute the weight of the barrier evenly to the N nearest girders, mating surfaces, or webs (GMSW’s). Nearest distance is measured from the C.G. of the barrier in a bridge cross section taken at mid-span. For cases when the weight of a barrier can be distributed to either of two GMSW’s that are equal distance left and right of the barrier C.G., and these GMSW’s are furthest from the barrier, the load will be distributed to the exterior-most GMSW. If the span contains 2N or fewer GMSW’s, the railing load will be distributed evenly to all GMSW’s.

Sidewalk and Pedestrian Live Load

Sidewalk and Pedestrian loads are distributed using a similar method and using the same N value as barriers. However, if the sidewalk is wider than N GMSW’s; the load will be distributed to all GMSW’s lying directly beneath the sidewalk. Hence, the definition is a follows:

Distribute the sidewalk weight and pedestrian live load evenly to the greater of: all girders, mating surfaces, or webs (GMSW’s) lying directly under the sidewalk; or the N nearest GMSW’s measured from centerline sidewalk using a bridge cross section taken at mid-span. For cases when the sidewalk weight can be distributed to either of two GMSW’s that are equal distance left and right of C.L. sidewalk, and these GMSW’s are furthest from the C.L. sidewalk, the load will be distributed to the exterior-most GMSW. If the span contains 2N or fewer GMSW’s, the load will be distributed evenly to all GMSW’s.

Overlay Load

Layout of Overlay:
The exterior edge of the overlay terminates at the interior-most of:

  1. Interior toe of Exterior Barrier

  2. Face of sidewalk (if no interior barrier)

  3. Interior toe of Interior Barrier

Two options are available for distributing overlay loads. The LRFD Specifications, Section 4.6.2.2.1, states that the overlay load may be evenly distributed over all the girders. If the uniform method is selected, the overlay load is computed as:

 w = (Wcc tolay golay) / Ngirders

where:

Wcc

Width of the deck from curb to curb. In PGSuper this is the distance between the insides of the traffic barriers.

tolay

Depth of the overlay

golay

Density of the overlay

Ngirders

Total number of girders

The second option is to apply the overlay load using the tributary area between girders. For exterior beams, the tributary width is measured perpendicular to the beam from the middle of spacing between the adjacent beam and the traffic curb.

When the overlay width varies along the length of the bridge, the overlay will be modeled with trapezoidal load segments.

Bridge Site Stage 3

Future Overlay Load

Future overlays are modeled the same as overlays that are constructed with the bridge. Future overlays are applied in Bridge Site Stage 3.

Live Load

The live load is applied to stage 3. The HL93 live load model will be used and applied between the centerlines of bearings as per the LRFD Specifications. 

Pedestrian Live Load

Pedestrian live load on sidewalks is computed based on LRFD 3.6.1.6. Load magnitude is based on the input sidewalk width and the load density input in the Bridge Site 3 Project Criteria tab. The pedestrian live load is distributed to girders using the same distribution as the sidewalk dead load. The pedestrian live load is applied concurrently with the vehicular live loads.
