Girder Description - Transverse Reinforcement

This tab of the Girder Editor defines transverse reinforcement.

Mild Steel Material

Select the appropriate mild steel for the transverse rebar

Roughening of Top Flange

Check the box to indicate the top flange is intentionally roughened to an amplitude of 0.25 inch to help resist horizontal interface shear.

Transverse Reinforcement Zones

The transverse reinforcement is described by zones, beginning at the left end of the girder. Transverse reinforcement can optionally be forced to be symmetrical about the center of the girder. Transverse reinforcement zones are described by their length, spacing and bar size.

If zones are made symmetrical about the girder centerline, right-most zone data is either automatically extended or truncated to mid-girder such that the zones are always symmetrical. Similarly, if zones are not symmetrical, right-most zones are either extended or truncated such that the last zone extends to the right end of the girder.

Primary bar zones contain vertical shear reinforcement that can optionally be extended into the deck for horizontal interface shear, and also be used for confinement steel and splitting resistance. These bars typically represent the main stirrups in a girder.

Use the Insert Row and Delete Row buttons to add and remove rows from the grid. To enter a primary stirrup zone,

  1. Press the Insert Row button to insert a new row immediately above the selected zone.

  2. Enter the length of the stirrup zone

  3. Enter the stirrup spacing

  4. Enter the bar size.

  5. Enter the number of bars (legs) to be used to resist vertical shear.

  6. Enter the number of legs to be extended into the deck to resist horizontal interface shear.

  7. Enter the bar size to resist confinement.

Additional Horizontal Interface Shear Zones

The bars in these zones are used to resist horizontal interface shear and are used in addition to any primary bars that have been extended into the deck. This zone layout is independent of the primary zones.

Additional Splitting Reinforcement

This data allows input for two identical zones placed at both ends of the girder to provide splitting resistance. These bars are used in addition to any splitting resistance provided by primary bars.

Additional Confinement Reinforcement

This data allows input for two identical zones placed at both ends of the girder to provide confinement resistance. These bars are used in addition to any primary confinement  bars.

Restoring Library Defaults

By pressing the Restore to Library Defaults button, the transverse reinforcement zones and bottom flange confinement steel values are set equal to those defined in the PGSuper library.  All current data is overridden.

Tip: Default values for the transverse reinforcement zones can be defined in the Girder Library entry. This helps engineering organizations define and adhere to standard transverse steel arrangements. WSDOT and TxDOT have performed parametric studies on each their standard girders and have created standard stirrup arrangements. PGSuper is used to evaluate the stirrup arrangement in a particular structure.

Tip: You can use the Girder Designer to automatically layout and design transverse reinforcement.
