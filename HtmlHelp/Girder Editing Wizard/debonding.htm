Girder Description - Debonding

This tab is used to define debonding of straight strands in the girder.

Debonded strands are shown in red in the schematic picture and in the Girder View.

This tab will only be shown if debondable straight strands exist in the currently selected Girder Library entry.

Use of the Debonding Grid:

  1. Click on the Insert or Delete buttons to add or remove debonding of the current strands.

  2. Select the strands to be debonded using the drop down combo box in the Strand column of the grid.

  3. Define the distance(s) from the end of the girder to the debond location by typing it into the Debond Length column of the grid.

Debond lengths cannot exceed one-half of the girder length.

Note: Straight strands that are shown in grey cannot be debonded. This happens when the strand is not allowed to be debonded in the girder library entry.

Symmetric Debonding

Select this option if debond lengths are the same on both ends of the girder. If not, the grid expands to allow different debond lengths at the left and right ends of the girder.
