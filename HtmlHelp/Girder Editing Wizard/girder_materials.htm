Girder Description - General

Girder Name

Select a girder from the drop down list. If the list is disabled, this indicates that the same girder is used for the entire bridge. To define each girder individually, click on the hyperlink text. The hyperlink text can also be clicked when the drop down list is enabled to assign the selected girder type to all girders in the bridge.

Girder Concrete Properties

Release Strength - f'ci

Input the release strength, f'ci, for the girder concrete.

Strength - f'c

This is the crushing strength for the material as defined by most common specifications. This strength is normally defined as the stress value at a strain of 0.003 at 28 days.

Eci

If checked, the release modulus of elasticity is directly input. This value overrides the computed value

Ec

If checked, the final (28 day) modulus of elasticity is directly input. This value overrides the computed value

More Properties...

Press this button to edit the detailed concrete properties include unit weight and aggregate size.

Slab Offset

Enter the slab offset for this girder. If a single slab offset is used for the entire bridge, or if the slab offset is defined span-by-span, click on the hyperlink text to enable girder-by-girder slab offset definition. If the slab offset is defined girder-by-girder, clicking on the hyperlink text will apply the current value to all the girders in this span or to the entire bridge depending on the slab offset mode when this dialog was opened.
