Select Girder for Editing

Each girder in your model is potentially different. For example, if you have a bridge on a horizontal curve, the girders will all be a different length. In this step of the girder editing wizard, you will select the girder that you want to edit.

Select Girder to be edited.

Using the drop down lists, select the girder that will be edited. 

Note: You can directly edit a girder by double-clicking your mouse on it in the Bridge Model View or the Girder View.
