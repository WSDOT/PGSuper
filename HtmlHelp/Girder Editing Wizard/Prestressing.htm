Girder Description - Strands

Use this tab to define the filling of prestressing strand locations and jacking forces for the girder.

Available strand locations and fill order are defined by the strand grid in the Girder Library entry.

Strand Details

Specify Filling of Strands Using

Three options are available here:

  1. Total Number of Permanent Strands. When this option is used, both straight and web strands (harped or straight-web) are filled sequentially using the global strand fill order (Fill #) defined by the Girder Library entry's strand grid.

  2. Number of Straight and Number of Harped; or Number of Straight and Straight-Web. Here, the number of straight and web strands are defined independently. Straight and web strand locations are filled sequentially using their relative orders in the Girder Library strand grid.

  3. Direct Selection of Strand Locations. This option enables filling of strands out of sequence. Click on the Select Strands button to bring up the Select Strands dialog.

Temporary strands are filled either sequentially or using direct selection depending on the option selected above.
 

See Girder Editing - Strands  for more information on available types of strands and strand editing options.

Calculate Jacking Force

When checked, the maximum jacking force is computed for the specified number of prestressing strands. The maximum jacking force is computed based on the allowable strand stress requirements of LRFD 5.9.3. If the box is not checked, enter the jacking force. User-selected jacking forces may not exceed ultimate strand strength.

Temporary Strand Usage

Temporary strands can be pretensioned or post-tensioned. Pretensioned strands are built into the girder and are effective during lifting and shipping. Post-tensioned temporary strands can be applied to the girder either immediately after lifting or immediately before shipping. Temporary strands are assumed to be removed after shipping. Temporary strands post-tensioned immediately before shipping are assumed to have no impact on camber and losses.

Use the drop down list to describe how temporary strands are used.

Vertical Location of Web Strands

If enabled by the current Girder Library Entry, the location of the entire web strand group (harped or straight-web strands) can be continuously adjusted up and down at the girder end and/or harping point locations. Several adjustment options are provided to make input easier. Enter the appropriate adjustment value.

  • Web strands must lie within the narrowest region of a girder's web.

  • Strands may only be adjusted within the vertical limits defined in the library. The current valid range is shown as an aid to help you select adjustment values.

  • Note that increasing the number of harped strands may decrease the size of the valid adjustment range.

  • Hold the CTRL key down when selecting the location datum and the current input value will be converted to the new basis of measurement.

Prestressing Strand Type

Select the grade, type, and size of the prestressing strand.
