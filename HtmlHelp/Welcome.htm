PGSuper (Prestressed Girder SUPERstructures) is Precast-Prestressed Girder design, analysis, and load rating software for the AASHTO LRFD Bridge Design Specification.

PGSuper is a software tool jointly developed by the Washington State and Texas Departments of Transportation and is a member of the Alternate Route Project.

PGSuper is licensed under the terms and conditions of the Alternate Route Open Source License. Please read this license carefully before using this software.

Getting Started

PGSuper documentation consists of four manuals that will help you use, manage, and understand this state of the art software. The Tutorial will get you up and running with PGSuper in no time. The User Guide provides in-depth coverage of the software's features and functionality. The Theoretical Manual details how PGSuper is performing its engineering tasks. The Administrator Guide explains how to install, customize, and use PGSuper in a production engineering environment.
