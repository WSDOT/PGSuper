Pier Details - Connections

The structural boundary conditions and the geometry at the ends of the girders are defined on this tab.

Connections are defined on both sides of the pier. The Back Side is at the end of the previous span. The Ahead Side is at the start of the next span.

Boundary Condition

The boundary condition defines the connectivity between the spans and the pier. Using the drop down list, choose one of the following options:

Hinged

There is no moment connectivity between the spans and piers

Roller

There is no moment connectivity between the spans and piers

Continuous Before Deck Placement

There is moment continuity between spans, but not with the pier. Continuity is achieved before deck placement

Continuous After Deck Placement

There is moment continuity between spans, but not with the pier. Continuity is achieved after deck placement

Integral Before Deck Placement

There is moment continuity between the spans and the pier. Continuity is achieved before deck placement

Integral After Deck Placement

There is moment continuity between the spans and the pier. Continuity is achieved after deck placement

 

NOTE: The boundary conditions at all piers are treated as hinged connections if the analysis mode is set to Simple Span. See Chapter 7 for more details.

Connection Definition

The connection definition describes the geometry at the ends of girders as well as the size of any end diaphragms.

Copy Connection Definition from Library

Connection definitions that are stored in the Connections Library can be used to quickly define the connection geometry. Press the button labeled "Copy Connection Geometry from Library" to select a pre-defined connection. The connection geometry will be copied from the library into this tab.

NOTE: Unlike other library entries, the connection data is not linked to the selected connection library entry.

Girder Connection Properties

The girder connection properties describe the location of the centerline of bearing relative to the Abutment/Pier Line, the location of the ends of the girder, and the size of the support area.

Bearing Offset

The bearing offset defines the distance from the Abutment/Pier Line to the Centerline of Bearing line. This distance may be measured normal to the Abutment/Pier Line or along the centerline of girders.

NOTE: When the bearing offset is measured along the centerline of the girders, and the girders are not parallel, there will be a unique centerline of bearing line for each girder.

End Distance

The end distance locates the end of the girder. This distance can be measured from either the Abutment/Pier Line or from the Centerline of Bearing line. This distance can be measured along the centerline of the girders or normal to the Abutment/Pier Line or the Centerline Bearing.

The end distance, along with the bearing to bearing span length, defines the overall plan length of the girder.

NOTE: The centerline of bearing line for each girder is always parallel to the Abutment/Pier Line.

Support Width

The support width defines the width of the bearing area and locates the face of support for shear calculations. Many of the LRFD specification requirements related to shear reference the face of support. PGSuper locates the face of support at one-half the support width from the centerline of bearing, measured along the centerline of the girder.

Diaphragm

The diaphragm data describes the size of diaphragms at the ends of a pier and how the dead load of the diaphragm is to be considered in the structural analysis.

Diaphragm Height and Width

Define the cross sectional area of the diaphragm by its height and width. The dead weight of the diaphragm is based on this cross sectional area, the tributary width of the girder, and the unit weight of the slab concrete.

Using the drop down list, specify how the dead load of the diaphragm is considered in the structural analysis.

Apply weight of diaphragm over CL Bearing

The weight of the diaphragm is applied directly over the Centerline of Bearing so that it creates only a dead load reaction.

Apply weight of diaphragm to girder

The weight of the diaphragm is applied directly to the girder so that it produces moment, shear, and reaction. Enter the distance from the Abutment/Pier Line to the center of gravity of the diaphragm. This is the location where the resultant of the diaphragm dead load is assumed to act.

Ignore diaphragm weight

The dead load of the diaphragm is not considered in the structural analysis.
