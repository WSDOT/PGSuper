Creating the Project

New PGSuper projects are created from project templates. The templates contain standard configurations for bridge geometry and components, appurtenances, loading, and design criteria. Since Example Bridge 1 is a WSDOT WF58G girder bridge, the easiest way to start is with the WF58G template.

TIP: You can customize project templates and their organization. See Chapter 10 of the PGSuper User's Guide for more information.

Start PGSuper

To start PGSuper, select Start | Programs | Engineering Applications | PGSuper.

Create a New Project

1. Select File | New. This brings up the New Project dialog.

2. Select the WF-Girders Project Type and the WF58G template. Press the OK button.

3. If the Project Properties dialog is displayed, you may enter some information about this project.
