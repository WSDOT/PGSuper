Adding the Pedestrian Live Load

Pedestrian live load can be added to this bridge model. The pedestrian live load is considered along with the vehicular live load when a combination railing is used. When there is only a sidewalk the pedestrian live load is considered separately from the vehicular live load and it is assumed that the vehicular live load can mount the sidewalk.

Select Loads | Live Loads to open the live load configuration dialog. In the Design Limit State group, check the HL-93 and Pedestrian on Sidewalk loadings.
