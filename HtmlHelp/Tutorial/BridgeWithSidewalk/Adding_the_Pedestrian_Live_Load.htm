Adding the Pedestrian Live Load

Pedestrian live load can be added to this bridge model. The pedestrian live load is considered along with the vehicular live load when a sidewalk is defined. Pedestrian loads can be enveloped or applied concurrently with vehicular live loads.

Select Loads | Live Loads to open the live load configuration dialog. In the Design Limit State group, check the HL-93 use the combo box below to apply Pedestrian on Sidewalk loadings concurrently with the HL-93 load.
