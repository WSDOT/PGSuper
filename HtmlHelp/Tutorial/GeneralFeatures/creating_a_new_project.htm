Creating a New Project

New PGSuper projects are created from project templates. The templates contain standard configurations for bridge geometry and components, appurtenances, loading, and design criteria. Since this is a WSDOT W58G girder bridge, the easiest way to start is with the W58G template.

TIP: You can create your own project templates and customize their organization. See Chapter 10 of the PGSuper User's Guide for more information.

Start PGSuper

To start PGSuper, select Start | Programs | Engineering Applications | PGSuper.

Create a New Project

1. Select File | New. This brings up the New Project dialog.

2. Select the W-Girders tab and the W58G.pgt template. Press the OK button.

3. This will bring up the Project Properties dialog where you can enter information about your project.

NOTE: The Project Properties dialog may not open when creating a new project. If this is the case, select File | Project Properties or Project |
Properties. Use the check box at the bottom of the dialog to control whether the dialog opens when new projects are created.

Finally the default configuration for this type of bridge will be displayed in the Bridge Model View.
