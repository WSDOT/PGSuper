Editing the Bridge Model

The bridge modeled in the default U54G6 template isn't exactly what we need. In the next few sections you will learn how to locate the bridge piers and establish the framing plan for the bridge girders.

We define the bridge beginning with the piers. The piers are where the bridge geometry intersects the roadway geometry (for this example, we'll just go with the default roadway). The locations of the piers establish the location of the bridge along the alignment.

Defining the Bridge

1) Select Edit | Bridge.... This will bring up the Bridge Description Dialog.

On the General tab, set the number of girder lines to 1.

2) Layout spans and piers

Select the Framing tab of the Bridge Description dialog. Press the Add Span button to create a new span, and thus a new pier. We could edit the stations of each pier, but it is sometimes easier to hold a pier at a given station and layout the rest of them by span length.

Press the Layout by Span Lengths... button and enter the information show. When you press the OK button, the pier stations will be adjusted for these span lengths.

3) Set pier 2 boundary conditions

This bridge is continuous at pier 2. To make the connection at pier 2 continuous after deck placement, select the "Edit Pier Details" button for pier 2.

Next, select the Connections tab and change the Boundary Condition  to Continuous after deck placement.

The connection geometry describes the location of the point of bearing relative to an arbitrary reference line and the location of the end of the girder relative to the point of bearing. Predefined connections are stored in the Connections Library. See Chapter 8 of the PGSuper User's Guide for more information.

4) Describe the Deck Geometry and Materials

Select the Deck Geometry and Materials tab of the Bridge Description Dialog and enter the information shown below.

Press the OK button. Notice that the Bridge Model View is updated to reflect the current project data.
