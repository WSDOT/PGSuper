User Defined Vehicular Live Loads

In addition to supporting the LRFD HL-93 live load (as defined in 3.6.1.3), PGSuper allows you to define your own live loads and envelopes of live loads. Live loads can be defined for two different load categories:

1) Design Limit States

  • Service I

  • Service IA (No longer applicable for LRFD 4th Edition with 2009 interim provisions and later)

  • Service III

  • Strength I

2) Fatigue Limit States

  • Fatigue I (Added in LRFD 4th Edition with 2009 interim provisions)

3) Permit Limit States

  • Strength II

The application of user-defined vehicular live loads is a two-step process. First you must create one or more Vehicular Live Load entries in the PGSuper library (these may be pre-defined by the library publisher), and then you must apply the loads by selecting them in the Specify Live Loads dialog.

Definition of Vehicular Live Loads

Vehicular Live Loads are defined in the PGSuper Library. Please read Vehicular Live Load Dialog description for detailed information on defining live loads.

Tip: If you are not familiar with using the library, a quick read of Introduction to Working with Libraries will help you to gain solid footing.

Specifying (Applying) Vehicular Live Loads

Once Vehicular Live Loads are defined in the library you can apply them selecting Loads | Live Loads (). This will bring up the Live Loads dialog.

Also, refer to Vehicular Live Loads section the in the Theoretical manual for more information.

Pedestrian Loads

PGSuper will automatically generate pedestrian loads on sidewalks if the loading is enabled. If your bridge has sidewalks, select how to combine the load with vehicular loads in the Design Limit State box.

You must explicitly define the live load for pedestrian only bridge. See the Pedestrian Only Bridge tutorial for details.
