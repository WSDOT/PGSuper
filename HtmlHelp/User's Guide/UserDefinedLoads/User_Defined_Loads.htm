User Defined Loads

Static Loads

There are three different types of user-defined static loads: Point (concentrated), linearly distributed loads, and moments. Loads are applied to individual girders directly along their flexible lengths, hence it is up to you to determine how to distribute loads to the girders.

Applying and Editing User-Defined Loads

User defined loads can be added to the model by selecting "Loads | Add Point Load", "Loads | Add Distributed Load", or "Loads | Add Moment Load" from the main menu. Loads can be edited from the User Defined Loads Editing view. This view can be brought up by selecting "Loads | Edit User-Defined Loads" from the main menu or by pressing on the toolbar. This view allows you to edit, delete, or add new loads to the model.

User-defined loads can also be created and edited using the Girder View

Basic Facts About User-Defined Loads

  • User-defined loads may be applied in any of the bridge site stages (Bridge Site 1, 2, 3) and a the effects of applied loads are carried cumulatively to subsequent stages.

    To add a load in one stage, and remove it in another, added an equal and opposite load in the removal stage

  • The loads must be assigned to either the LRFD DC or DW dead load cases or can be added directly to the HL-93 response in the LL+IM live load case.

  • Loads that are designated as LL+IM are static loads that are added to Service and Strength I limit states. These loads are different than the user-defined vehicular loads described below.

  • User-defined loads placed in the "LL+IM" load case are be added directly to the HL-93 live load response envelope after distribution factors have been applied.

  • Since they are assigned to LRFD load cases, the responses due to user-defined loads are added into the LRFD limit states using the appropriate factors.

  • Positive load values are applied in the direction of gravity. Positive moments are counter clockwise, following the right hand rule.

  • Load locations are measured from the left girder support and must lie within the flexible span length of the girder.

  • If any part of a user-defined load is placed outside of the flexible span length of the girder it is applied to, the entire load will be ignored. A message will be posted to the Status Center to inform you of this.

  • Fractional location values may be specified. The value must range from 0.0 to 1.0. For example a load applied at a fractional location of 0.25 is placed at the first quarter point of the flexible span length.

  • Effects due to user-defined live loads are NOT included in the LRFD live load deflection analysis.

  • User-defined moments can only be applied at the start and end of the span.
