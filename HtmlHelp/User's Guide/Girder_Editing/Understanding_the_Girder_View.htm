Understanding the Girder View

The Girder View provides a graphical representation of a girder. The view is divided into three regions, the Control Bar, the Elevation View, and the Section View.

Control Bar

The Control Bar is located at the top of the Girder View. The Control Bar is used to control the contents of the Girder View.

Selecting a Girder

To select a girder to view:

  1. Select a span from the drop down list

  2. Select a girder from the drop down list

Check the Synchronize with Bridge Model View selection to keep the same girder selected in both views.

Selecting a Stage

Use the drop down list to select the stage in which user-defined loads are to be shown

Selecting a Cross Section

There are several methods for selecting a cross section in the Girder View. From the Control Bar, you can select a cross section by pressing the Section Cut Offset button. Pressing this button displays the Section Cut Location dialog.

Using the option buttons, select a predefined section location, or select the last option button and enter a specific location. Section locations are measured from the left end of the girder.

Adding and Editing User-Defined Loads

Drag the point load, distributed load, or moment load icon from the control bar onto the girder at the location near where you want to apply the load. Loads may then be edited by double-clicking on them.

The Legend helps define the load cases that user-defined loads are applied in. You can move the legend around the view by dragging it and you can change the number of rows shown in the legend by right-clicking on it.

Elevation View

The Elevation View shows the a schematic representation of the girder elevation. This view shows the bearing points, span length, girder length, location of harping points, prestressing strands, longitudinal reinforcing steel, and the C.G. of the prestressing force.

Selecting a Cross Section

The Elevation View offers the easiest method for selecting a girder cross section. Simply drag the Cross Section Selection Tool to the location where you wish to exam the girder's cross section.

8 Alternatively, you can use the Context Menu to select the cross section.

Context Editing

You can quickly edit any of the girder attributes by selecting an attribute category from the context menu. The context editing commands will take you directly to the appropriate tab in the Girder Editing dialog.

Section View

The Section View displays the a scaled representation of the girder cross section. This view shows the girder cross section at the section cut, the location of the prestressing strands and longitudinal reinforcing steel, the C.G. of the prestressing force, and the height and width dimensions of the girder. The same context editing capabilities are available in the Section View as in the Elevation View.

Customizing the Girder View

The Girder View can be customized by enabling and disabling the various options. To customize the view select View | View Settings | Girder View (ALT,V,V,G). This will bring up the View Settings dialog.

On the Elevation View tab, enable or disable the various view attributes. If the Schematic option is enabled, the girder will be drawn as a schematic, otherwise, it will be drawn to accurate proportions.

On the Section View tab, enable or disable the various view attributes.
