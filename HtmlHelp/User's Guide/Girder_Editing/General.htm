General

The girder type and concrete properties are defined on the General tab of the Girder Details dialog.

The list of girders is disabled when the same girder is used for the entire bridge. You can enable span-by-span girder definition and select a girder name by clicking on the underlined (hyperlink) text. Similarly, you can use this girder type for the entire bridge by again clicking on the hyperlink text.

The concrete properties are the release strength, f'ci, the final strength, f'c, and the associated moduli of elasticity. If the check box in front of Eci and Ec are check, you can enter your own standard value otherwise PGSuper computes modulus of elasticity based on the concrete parameters.

Addition concrete details can be defined. Press the More Properties button to access these details.

The slab offset values are disabled when the same slab offset is used for the entire bridge or for all girders in this span. You can define the slab offset for this individual girder by clicking on the underlined (hyperlink) text.
