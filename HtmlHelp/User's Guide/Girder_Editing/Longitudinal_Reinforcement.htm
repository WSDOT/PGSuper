Longitudinal Reinforcement

The Longitudinal Reinforcement tab is used to describe longitudinal rebar in the girder. Each row in the grid defines a row of rebar described by its location along the girder, size, quantity, horizontal spacing (center to center) and cover location in the cross section. Its material type is also specified here.

Refer to the  Longitudinal Reinforcement in Girders section in the Theoretical Manual for a discussion of how, and when, girder longitudinal rebars are used in analysis.

Tip: Longitudinal reinforcement values are copied from the library whenever a new project is created or a new girder type is selected from the library. You can also replace the current rebar data with data defined in the library by pressing the "Restore to Library Defaults" button.
