Longitudinal Reinforcement

The Longitudinal Reinforcement tab is used to describe longitudinal rebar in the girder. Each row in the grid defines a row of rebar described by its size, quantity, horizontal spacing (center to center) and cover location in the cross section. Its material type is also specified here.

Tip: Longitudinal reinforcement values are copied from the library whenever a new project is created or a new girder type is selected from the library.
