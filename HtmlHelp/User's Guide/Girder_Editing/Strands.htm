Strands

Use the Strands tab on the Girder Details dialog to describe the filling of strands, jacking forces, type of strands, and vertical adjustment of harped prestressing strand locations.

Describing Prestressing Strands in a Girder

Allowable locations for strands in a given girder are defined in the strand grids in the Girder Library.

Pretensioned prestressing is described by filling allowable locations of straight, harped (or straight-web), and temporary prestressing strands, and the jacking force imparted onto the strands. Locations of harped (or straight-web) strands can be further defined by vertical adjustment of the strand pattern at the end of the girder or at the harping point.

Basic Types of Strands

 PGSuper provides input for three basic types of strands:

  1. Straight Permanent Strands can be placed anywhere within a girder section and are straight along the entire girder. "Permanent" means that these strands are jacked in the casting yard with the prestressing remaining for the life of the girder.

  2. Web Strands are permanent strands that can only be placed within the thinnest portion of a girder section's web(s). Web Strands have the unique capability to be vertically adjusted within the web region. There are two possible types of Web Strands: Harped Strands and Straight-Web strands. Harped strands are draped at the girders harping points, and Straight-Web strands are straight along the entire length of the girder.

  3. Temporary Strands are straight strands that can be placed anywhere within a girder. Temporary strands can be pretensioned or post-tensioned. Pretensioned strands are tensioned prior to casting. Post-tensioned temporary strands can be applied to the girder either immediately before lifting, immediately after lifting, or immediately before shipping. All Temporary Strands are removed after shipping (before Bridge Site Stage 1). Temporary strands post-tensioned immediately before shipping are assumed to have no impact on camber and losses.

Filling of Prestressing Strands

There are two basic methods for filling prestressing strand locations: 1) Enter a number strands - Strands are filled sequentially using the order defined by the pertinent girder library entry. 2) Direct selection of available strand locations using the "Select Strands" dialog. This allows strands to be filled out of order.

Entering Number of Strands (Sequential Strand Fill)

For this method, Strands are filled sequentially in the order defined by the permanent strand fill order in the girder's library entry. Strand 1 goes in position 1, strand 2 goes in position 2, and so on.

To enter the number of strands, simply use the Up/Down arrow buttons on the dialog to dial-in the desired number of strands. The number of strands is limited to the potential strand locations defined by the Girder Library Entry assigned to the girder in question. The number of strands may be specified as the Number of Straight and Number of Harped (or Straight-Web) or as the Total Number of Strands.

Direct Selection of Strand Locations

This method allows filling of strands by selecting individual strand locations in the "Select Strands" dialog.

Note: The automated design algorithm always fills strands using sequential strand fill.

Entering Jacking Forces

You can explicitly enter a jacking force for each group (straight, web, or temporary) of prestressing strands, or you can let PGSuper compute the maximum allowable jacking force based on LRFD 5.9.3. Explicitly entering the jacking force is useful when evaluating a design or existing structure.

To explicitly enter the jacking force:

  1. Remove the check from "Calculate Jacking Force"

  2. Enter the jacking force

To let PGSuper compute the maximum allowable jacking force

  1. Check the Calculate Jacking Force option.

Note : PGSuper will use the average jacking stress for all strands in the section when performing calculations.

Vertical Location of Web (Harped or Straight-Web) Strands

If allowed by the library entry; you can adjust the vertical location of harped strands at the end of the girder or at the harping point, or straight-web strands along the girder to control stresses. For example: In the casting yard stage, if there is excessive compressive stress at the top of the girder, near the ends, the web strands could be lowered to relieve the stress. Likewise, if there is excessive compressive stress at the bottom of the girder, between harping points, the harping points can be raised to relieve the stress. The basis location of the web strands can be described by:

  • Distance between the CG of the Web Strand Force and the Top of the Girder

  • Distance between the CG of the Web Strand Force and the Bottom of the Girder

  • Distance between the Top-Most Web Strand and the Top of the Girder

  • Distance between the Top-Most Web Strand and the Bottom of the Girder

  • Distance between the Bottom-Most Web Strand and the Bottom of the Girder

  • Eccentricity of the Web Strand Group as measured on the non-composite section

Note: Web strands must lie within the vertical bounds defined by the adjustment bounds defined in the girder library entry.

Note: Hold the CTRL key down while changing the basis of measurement and the input value will be converted to the new basis.
