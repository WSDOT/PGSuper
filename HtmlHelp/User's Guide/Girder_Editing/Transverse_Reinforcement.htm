Transverse Reinforcement

Transverse reinforcement is described using zones containing constant stirrup size and spacing. Zones are laid out from left to right along the girder. Zones can optionally be symmetrical or not-symmetrical about mid-girder.

Primary bar zones contain vertical shear reinforcement that can optionally be extended into the deck for horizontal interface shear, and also be used for confinement steel and splitting resistance. These bars typically represent the main stirrups in a girder.

If zones are made symmetrical about the girder centerline, right-most zone data is either automatically extended or truncated to mid-girder such that the zones are always symmetrical. Similarly, if zones are not symmetrical, right-most zones are either extended or truncated such that the last zone extends to the right end of the girder.

Theoretical Note: Stirrup zones in PGSuper are not meant to define the exact placement of individual physical stirrups, such as in a bridge plan set. PGSuper's stirrup zones define a region with a given bar size / legs / spacing for analytical purposes (e.g., a "smeared" stirrup model). This means that if an analysis location lies within a stirrup zone, Av/S information is taken directly from that zone. It is the responsibility of the engineer to determine precisely how this information maps into a detailed plan stirrup layout. Hence, zone lengths do not have to be an exact multiple of bar spacing.

Check the box to indicate the top flange is intentionally roughened to an amplitude of 0.25 inch.

Tip: Default values for the transverse reinforcement zones can be defined in the Girder Library entry. This helps engineering organizations define and adhere to standard transverse steel arrangements. WSDOT has performed parametric studies on each of its standard girders and has created standard stirrup arrangements. PGSuper is used to evaluate the stirrup arrangement in a particular structure.

To enter a primary stirrup zone,

  1. Press the Insert Row button to insert a new row immediately above the selected zone.

  2. Enter the length of the stirrup zone

  3. Enter the stirrup spacing

  4. Enter the bar size.

  5. Enter the number of bars (legs) to be used to resist vertical shear.

  6. Enter the number of legs to be extended into the deck to resist horizontal interface shear.

  7. Enter the bar size to resist confinement.

Additional Horizontal Interface Shear Zones

The bars in these zones are used to resist horizontal interface shear and are used in addition to any primary bars that have been extended into the deck. This zone layout is independent of the primary zones.

Additional Splitting Reinforcement

This data allows input for two identical zones placed at both ends of the girder to provide splitting resistance. These bars are used in addition to any splitting resistance provided by primary bars.

Additional Confinement Reinforcement

This data allows input for two identical zones placed at both ends of the girder to provide confinement resistance. These bars are used in addition to any primary confinement  bars.

Restoring Library Defaults

The input for transverse reinforcement can be tedious work. If, after modifying the transverse reinforcement, you want to restore it to the library defaults, simply press the Restore to Library Defaults button.

Tip: PGSuper can check if stirrup zone lengths are compatible with stirrup spacing and issue a warning if they are not. See the Project Criteria Limits and Warnings Tab for more information.
