Transverse Reinforcement

Transverse reinforcement is described in zones where the stirrup size and spacing is constant throughout the zone. The zones are described from left to right up to the girder mid-span. The reinforcement is assumed to be symmetrical about the mid-span point.

The default values for the transverse reinforcement step are defined in the Girder Library entry. This allows engineering organizations to define standard transverse steel arrangements. WSDOT has performed parametric studies on each of its standard girders and has created standard stirrup arrangements. PGSuper is used to evaluate the stirrup arrangement in a particular structure.

Transverse reinforcement can be vertical, horizontal, or both. The vertical reinforcement is always considered when evaluating the vertical shear capacity of a girder. Either the vertical or horizontal reinforcement is considered when evaluating splitting resistance in the pretensioned anchorage zone (LRFD 5.10.10.1). The vertical reinforcement is used for I-girders, bulb tee girders, and stemmed girders (double tee, ribbed girders, etc). The horizontal reinforcement is used for solid or voided slabs. For pretensioned box or tub girders the horizontal reinforcement will be used if the girder is taller than it is wide and the vertical reinforcement will be used if the girder is wider than it is tall.

Tip: Transverse reinforcement values are copied from the library whenever a new project is created or a new girder type is selected.

Transverse Reinforcement Zones

Transverse reinforcement is described in zones. The first zone begins at the left end of the girder, the next zone begins after the first zone, and so on to the middle of the girder. To enter a stirrup zone,

  1. Press the Insert Row button to insert a new row immediately above the last zone.

  2. Enter the length of the stirrup zone

  3. Enter the stirrup spacing

  4. Enter the bar size.

  5. Enter the number of bars per web.

Check the box to indicate the stirrups extend into the composite deck, are adequately developed, and contribute to the horizontal interface shear capacity of the section.

Check the box to indicate the top flange is intentionally roughened to an amplitude of 0.25 inch.

Top Flange Horizontal Interface Shear Reinforcement

The top flange horizontal interface shear steel are stirrups located in the top flange of the section only which protrude up into the slab. They are used to add strength to the slab-girder interface to resist horizontal shear forces. These stirrups are spaced evenly along the entire length of the girder. Each stirrup provides two legs of reinforcement.

NOTE: This input was more important for early versions of the AASHTO LRFD Bridge Design Specifications when 4 stirrup legs were required to cross the flange/slab interface boundary for wide top flanges.

Bottom Flange Confinement Reinforcement

The bottom flange confinement steel is used to check LRFD 5.10.10.2. To describe the confinement steel in the bottom flange, select the bar size, and the last zone that contains the confinement steel. The spacing of the confinement steel will match that of the stirrups zones.

Restoring Library Defaults

The input for transverse reinforcement can be tedious work. If, after modifying the transverse reinforcement, you want to restore it to the library defaults, simply press the Restore to Library Defaults button.
