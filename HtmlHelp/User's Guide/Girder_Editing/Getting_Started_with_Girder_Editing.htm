Getting Started with Girder Editing

Girder editing is accomplished with the Girder Details Dialog. To open the dialog, select Edit | Girder (ALT,E,G) or press the Edit Girder toolbar button .

The first step is to select the girder to edit. Use the drop down lists to select a span and girder

TIP: Avoid this step by double clicking on a girder in the Bridge Model or Girder View. This will open the editing dialog for the girder that was clicked on
