Copying Girder Properties

As you can see from the other sections in this chapter, it takes quite a bit of work to define a girder's design parameters. If every girder in your bridge is the same, you don't want to run through the Girder Editing Wizard for each one.

The Copy Girder Properties tool makes it easy to copy the properties from one girder to the other girders in your structure.

To copy girder properties:

  1. Select Edit | Copy Girder Properties (ALT,E,C)

  2. Select the girder from which the properties will be copied

  3. Select the girder(s) to which the properties will be copied

  4. Using the check boxes, select which girder properties will be copied.

  5. Press the Copy Now button to copy the properties.

  6. Repeat steps 1-4 until you are done copying properties, then press the Close button.

Tip: You can use the Girder Comparison Report to compare the properties of all girders in the bridge. The Girder Comparison Report is available from the standard reporting dialog.
