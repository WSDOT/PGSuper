Materials

The concrete and strand materials are defined on the Materials tab

Girder Concrete Properties

Release Strength - f'ci

Input the release strength, f'ci, for the girder concrete.

Strength - f'c

This is the crushing strength for the material as defined by most common specifications. This strength is normally defined as the stress value at a strain of 0.003 at 28 days.

Eci

If checked, the release modulus of elasticity is directly input. This value overrides the computed value

Ec

If checked, the final modulus of elasticity is directly input. This value overrides the computed value

Density for Strength

This is the density which is used along with f’c to calculate the modulus of elasticity if it is not overridden.

Density for Weight

This is the density used to determine the self-weight of the slab. The density should include the expected average effects of all reinforcement and prestressing in the structure.

Max Aggregate Size

Enter the maximum dimension of the concrete aggregate. This value is used to check minimum reinforcement spacing requirements per LRFD 5.10.3.1.2.

K1

Aggregate source correction factor per LRFD 5.4.2.4

Copy Properties from Library Entry...

Press this button to copy material properties from a pre-defined concrete material in the library. Note that you must still input the release strength manually since it is not defined in library entries.

Prestressing Strand Type

To describe the prestressing strand type

  1. Select the strand grade (SI : Grade 1725 or Grade 1860, US : Grade 250 or Grade 270)

  2. Strand type (Low Relaxation or Stress Relieved)

  3. Using the drop down list, select the strand size.
