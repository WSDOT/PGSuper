Multi-Girder Select Grid

This grid control allows you to select multiple girders in your PGSuper bridge model. Simply use your mouse to select girders by checking the boxes in the grid.

The grid works much like an Excel spreadsheet and supports the following features:

Multi-cell selection:

  • Ctrl-click or Ctrl-drag - to select multiple cells by mouse

  • Shift-click or Shift-drag - to select multiple contiguous cells by mouse

  • Select entire rows by clicking the row header

  • Select entire columns by clicking the column header

  • Select all values by clicking on the upper left cell

  • A mouse click on a selected cell will toggle all selected check boxes, after which cells will become un-selected.

  • Selection is turned off by clicking the mouse on an unselected cell

  • Keyboard Manipulation in Grid:

  • Use arrow keys to navigate between cells. Holding Shift enables multi-selection.

  • Tab key – Next dialog control

  • Space bar to toggle check boxes
