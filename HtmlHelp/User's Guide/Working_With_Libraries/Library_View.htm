Library View

The PGSuper Library is found in the Library View. To activate the Library View, select Library | Edit Libraries (ALT,B,E ). The Library View is shown below.

In the left pane, you see the list of libraries. By selecting one of the Folder Icons, you see the contents of the library in the right pane.

The library entries are annotated with icons to indicate their status. The lock icon () indicates the library entry is in the Master Library and cannot be changed. The check icon () indicates that the library entry is currently in use.

Tip: You cannot delete entries that are in use.
