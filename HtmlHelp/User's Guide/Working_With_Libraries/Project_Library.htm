Project Library Entries

Project Library Entries are specific to your PGSuper Project. You can create, edit, and delete Project Library Entries (you cannot delete entries that are currently in use). Project Library Entries are stored locally in your project file and remain a part of your project until you remove them.

So why have a Project Library? If you are designing typical structures for an organization that has developed standard PGSuper libraries, you will almost certainly want to use their Master Library for your project. However, there are cases where you will want to create library entries that are local to your project only:

  • Pioneering the use of PGSuper for an organization.

  • Design of new, non-standard girder types

  • Unique structures

Tip: If you find yourself creating the same library entry over and over again for different projects, it is likely that it is a good candidate for the Master Library. Contact your PGSuper Administrator and ask to have it placed in the Master Library so it can be shared by you and others in your organization.
