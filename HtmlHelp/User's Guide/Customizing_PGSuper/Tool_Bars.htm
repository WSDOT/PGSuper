Tool Bars

There are two methods for selecting which toolbars are visible and which ones are not. The first method is to position the mouse over the toolbar area at the top of the main window and press the right mouse button. This will cause a small menu to appear that contains the names of the toolbars. Simply select the toolbars you would like to be visible. The second method is to use the Toolbars dialog. Select View | Toolbars (ALT,V,T) to display the dialog. Indicate which toolbars are to be visible by checking the box next to the toolbar's name. With this dialog, you can turn tooltips on or off as well.
