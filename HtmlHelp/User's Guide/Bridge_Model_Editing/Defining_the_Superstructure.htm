General Bridge Description

Editing of the entire bridge can be done from the Bridge Description dialog. The bridge description starts with a general description of the superstructure.

General Description of Bridge

The General tab shown above is used to define the primary characteristics of the superstructure.

To describe the superstructure:

  1. Select the Girder Family from the drop down list. The girder family is the general type such as I-Beam, Voided Slab, Box Beam, etc. This parameter is usually set by the project template and doesn't need to be changed.

  2. Select the Girder Type from the drop down list. This list contains the various types of girders defined in the Girder library.

  3. Use the check box below Girder Type to indicate if a single girder type is used for the entire bridge or if a unique girder type will be assigned to every girder in the bridge.

  4. Enter the number of girders.

  5. Use the check box below Number of Girders to indicate if the same number of girders is used in every span or if the number of girders is defined span by span.

  6. Select the Spacing Type.

  7. Enter the girder spacing

  8. Position the girders transverse in the cross section
    Enter the distance from the CL Bridge or the Alignment to a specific girder or the center of the girders (mid-point between the exterior girders)
    See figure below

  9. Select the datum from which the girder spacing and transverse location is measured.

  10. Select the girder connectivity of adjacent girders.

    Note that this only applies for adjacent girders (e.g., No Deck and CIP Deck Overlay deck types). The connectivity value, the Deck Type, and the method for which distribution factors are to be computed (in the Project Criteria Library); are used to determine the equations to be used to compute live load distribution factors. Refer to section 4.6.2.2 of the AASHTO LRFD Specifications for more information.

  11. Select the Girder Orientation. This orients the girder in the cross section. I-Beams are typically plumb while U-Beams, Deck Bulb Tees, and Slabs are typically normal to the deck.

  12. Select the Deck Type. The available deck types depend on the type of girder selected.

A description of the bridge is given on the right side of the tab.

Girders are positioned transversely by giving the distance from the Bridge Line or the Alignment to a specific girder
