Designing a Girder

This section will discuss the mechanics of designing a girder. In practice, designing a girder can be very complex and tedious. In many cases, PGSuper does all of the hard work.

Using the Preliminary Designer

To use the Preliminary Girder Designer:

  1. Start the design by selecting Project | Preliminary Girder Designer (ALT,P,D) (Ctrl+D,). This will display the following dialog:

  2. Select girder(s) to design. Use the radio button to design a single girder, or multiple girders.

  3. Select your design options. PGSuper can design for flexure, shear, or both.
    Note that, if applicable, you can select to keep the Slab Offset constant during design, or have the designer compute an optimal offset.

  4. Press the OK button to start the design

When the design finishes, you will be presented with the Design Outcome Dialog.

Tip: Stirrup layouts for WSDOT standard girders are specified by the WSDOT Standard Plans. Hence, when designing WSDOT standard girders, you will typically perform only a design for flexure. Designing for shear will cause the standard stirrup configurations to be replaced with PGSuper's guess.

Tip: You can quickly design a girder by selecting it in the Bridge Model View, right clicking on it, and selecting Design Girder from the context menu

Tip: Availability of the option to design slab offset is controlled on the Project Criteria library entry's Design Tab

Reviewing the Design Outcome

Before you accept PGSuper design, it is prudent to review the design outcome. A summary of the design outcome is displayed in the Design Outcome Dialog. The design outcome report shows a comparison between the values for the parameters in the current girder verses the ones the designer has guessed.

Accepting/Reject a Design

After reviewing the design details, you must decide to accept or reject the Designer's proposed design. Click on the appropriate button close the dialog and Accept or Reject the design. If you accept the design, the current input parameters will be replaced with those that the Designer has determined. If you reject the design, the parameters remain unchanged.

Tip: Always run a full specification check (Details Report, or Spec. Check Report) after you run a design. Approximations made by the design tool may not satisfy all specification provisions.
