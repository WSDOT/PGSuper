Chapter 10 - AutoCalc Mode

You probably noticed that PGSuper updates reports and graphs without you having to issue an update command or tell PGSuper to do run specification check. What you are experiencing is AutoCalc (automatic calculation) mode. You simply create graphs and reports and PGSuper does whatever work is necessary to satisfy these requests. When the bridge model changes, PGSuper automatically updates any analysis results that are being displayed. This is part of the Bridge-centric user interface. When the AutoCalc mode is turned on, you can focus on the bridge and the analysis, not the commands needed to produce output.
