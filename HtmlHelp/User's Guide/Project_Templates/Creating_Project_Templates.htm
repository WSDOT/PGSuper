Creating Project Templates

Creating project templates is an easy task. The hard part is coming up with good templates.

To create a project template:

  1. Create a bridge model using the editing techniques described in Chapter 3 and Chapter 4.

  2. Save the bridge model as a template. Select File | Save As Template (ALT,F,E).

That's all there is to it.

In order to use your project template, it must be stored in a Libraries and Template Package. And this brings us to the next topic, Managing Project Templates.
