Managing Project Templates

In order for your templates to be available in PGSuper, they must be stored in the User Templates Folder. Customizing the User Templates Folder is discussed in Chapter 12. In this section, you will learn how the User Templates Folder relates to creating new PGSUper Projects and the New Project Dialog.

Organizing Your Templates

Your templates must be stored in the User Templates Folder or in a folder immediately below the User Templates Folder. In the figure below, the user templates folder is named "My PGSuper Templates" - you can name your folder whatever you want.  By creating sub-folders, you can organize your templates. The default templates created for WSDOT and provided with PGSuper are organized by girder type. Other organizations might include geographical region, span lengths, or state standards.

All of the templates in the User Template Folder are presented in the General tab of the New Projects dialog. Each of the sub-folders corresponds to a tab in the New Projects dialog.
