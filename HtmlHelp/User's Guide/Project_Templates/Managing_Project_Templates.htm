Managing Project Templates

In order for your templates to be available in PGSuper, they must be stored in a libraries and templates package (LTPackage) and published on a PGSuper Catalog Server. The easiest type of LTPackage is a folder on your hard disk. In this section, you will learn how create a LTPackage, how it relates to the New Project Dialog, and how to configure PGSuper to use your LTPackage.

Master Library File

A LTPackage must have a master library file. This is a PGSuper Library File that will be referenced when projects are opened or created. You can create a Master Library project and build your own library if you like. For convenience, a default master library file is installed with PGSuper. This file is C:\Program Files\WSDOT\PGSuper\WSDOT.lbr. We'll use this file to build a LTPackage.

Organizing Your Templates

Your template files are stored in file system folders on your computer. Use can use the folder names and tree structure to provide organization to your templates. The tree structure is replicated in the New Project Dialog (See below).

PGSuper Catalog Servers

PGSuper Catalog Servers provide a listing of LTPackages. Catalog servers can be FTP or HTTP servers on the Internet or a Master Library File and Templates Folder on your local or network file system. For the sake of this example we'll use the local file system as a PGSuper Catalog Server.

To create a catalog server for your LTPackage (which is nothing more than a Master Library File and a Templates Folder),

  1. Select File | Configure PGSuper

  2. Select the Use libraries and templates published on the local file system, network, or the Internet option.

  3. Press the Add/Edit button to add a catalog server

    1. In the PGSuper Catalogs Servers dialog, press the Add button to add a new catalog server

    2. Define the catalog server

    3. Press OK to close the PGSuper Catalog Servers dialog

  4. Select the "MyCatalogServer" and the "MyCatalogServer" LTPackage

  5. Press the Update Library and Templates Now button to configure PGSuper with your LTPackage

When you create a new PGSuper Project, your templates will be listed under PGSuper Project Templates - Workgroup Templates.
