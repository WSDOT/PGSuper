Girder Connection Error

"Error - The end of the girder is located off of the bearing at..." is posted to the Status Center

PGSuper tries to prevent user input that causes geometrical errors. However, this error can be caused by a combination of inputs and cannot be trapped at input time. This error typically occurs when there is a large skew, and the End Distance in the connection library entry is very close to the bearing Offset (i.e., the girder cantilever is very small).

For example: In the figure below, with a Bearing Offset = 9” measured along the alignment; and an End Distance =9” measured from CL pier, normal to CL pier. For this case a zero skew would place the bearing at the end of the girder, a valid condition. However, any non-zero skew would cause the bearing to be beyond the end of the girder – negative cantilever - Not Good: we cannot analyze girders supported by nothing but air....

The issue can be fixed by increasing the end distance, or reducing the girder skew angle going into the pier.

Tip: The end distance can be increased by editing the currently selected Connection Entry directly, or by selecting a different connection entry at the pier in question.
