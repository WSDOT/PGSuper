Analysis Results View

The Analysis Results View is used to display the structural analysis results. The Analysis Results View consists of a Control Bar and a graph.

Selecting Analysis Results

Using the drop down lists on the control bar you can select the analysis results that will be displayed.

Span/Girder Lists

Select a girder for which results will be displayed

Stage

Select the analysis stage

Loading

Select a load, load case, or limit state

Actions

Select a structural action

Multiple loading conditions can be selected. To select several individual loads, hold down the Ctrl key and click on the loading name. To select a sequence of loads, hold down the Shift key and click on the first and last loading in the sequence. A sequence can also be selected by selecting the first loading with the mouse, dragging the mouse of the load list while the left button is depressed, and release the mouse button over the last load in the sequence. 

Analysis Type

Select the analysis type for which results are displayed. The default is the structural analysis method defined in the Project Criteria.

Simple Span

All spans are assumed to be simple spans, in all stages, regardless of boundary conditions

Simple Spans Made Continuous

Spans are simply supported until continuity

Simple/Continuous Envelope

Envelope of Simple Span and Simple Spans
