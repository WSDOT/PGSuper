Report Views

Report Views display the various reports available in PGSuper. Reports consist of chapters that may be included or excluded at your discretion.

Reports

PGSuper can create several different reports for you. Some of the available reports are listed below.

Bridge Geometry Report

A report of deck elevation, top of girder elevation, haunch depth, and other related geometric information

Details Report

A detailed report of inputs, outputs, and intermediate calculations.

Fabrication Optimization

A report listing various combinations of pretensioned and post-tensioned temporary strands and required release strength.

Girder Comparison Report

Compares the properties of all the girders in the bridge

Hauling Report

Reports hauling analysis calculations

Lifting Report

Reports lifting analysis calculations

Spec Check Report

Reports the results of all specification checks

Summary Report

A terse report containing summary input and output information.

Additional reports are available by installing PGSuper extensions.

Displaying a Report

To display a report:

  1. Select View | Reports | <report name>. This will display a report configuration dialog that can be used to customize your report.

  2. Input the reporting parameters. Reports often require span and girder number as well as a list of chapters. Scroll through the list of chapters and place a check mark next to the ones you want included in the report. You can select several chapters at a time using the standard Ctrl+Click and Shift+Click mouse operations.

  3. Press the OK button to create the report. The Report View is displayed when PGSuper finishes creating the report.

Tip: Quickly display a report by selecting a girder in the Bridge Model View, right click, and choose a report from the context menu. Note that only favorite reports are available from the girder view.

Editing a Report

After a report has been created, you can edit the contents by pressing the Edit button in the Report View window or selecting Edit from the report context menu. Editing will display the report configuration dialog used when the report was created.

Reviewing a Report

To review a report, simply use the scroll bar to move up and down through the report. If you are using Internet Explorer 5 or later, you can search the report for keywords.

To search the report, click the right mouse button over the report window and select the Find option. Enter the keyword or phrase to search for in the Find dialog, .

Tip: The Table of Contents feature makes it easier to navigate reports. To view the Table of Contents, click the right mouse button over the report and select Table of Contents.

Tip: You can search through a report by right-clicking to bring up the context menu and selecting Find

Printing a Report

To print a report, select File | Print (ALT,F,P) (Ctrl+P,). Printing is controlled by the Internet Explorer Browser. PGSuper does not adjust the margins or the header and footer information. We recommend you set Internet Explorer's margins to 0.5" and to print the page number in the header.  For some reports it may be useful to print in Landscape mode.

Tip: Reports can be printed to a file using a tool like the freely-available PDFCreator.

Making a List of Favorite Reports and Creating Custom Reports

The list of reports in PGSuper can be quite long, and after a while you might find that you really only use a few of them in your daily work.
