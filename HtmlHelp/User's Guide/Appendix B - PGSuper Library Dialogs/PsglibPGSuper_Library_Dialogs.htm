Appendix B - PGSuper Library Dialogs

The PGSuper Library Dialogs allow you to enter commonly used information that can be later used to describe information in a PGSuper Project . Each of the dialogs is designed to allow you to easily enter information about its entry in the currently selected system of units. Data range and error checking is performed to keep erroneous information from getting into the library. However, this error checking is not perfect and it is the responsibility of the engineer (you) to input data which is practical for real bridge structures.

The following types of entries are supported by the library:

Concrete

Connections

Girders

Traffic Barriers

Project Criteria

Load Rating Criteria

Vehicular Live Load
