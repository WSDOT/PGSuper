Concrete Material Dialog

The Concrete Material Dialog allows you to edit the properties of concrete materials stored in the library. PGSuper considers all concrete to be Normal Weight Concrete in accordance with LRFD 5.2. Concrete definitions that do not conform to LRFD 5.2 are treated as if they were normal weight concrete. The properties that may be specified for a concrete entry are as follows:

Entry Name

Each concrete material entry must have a unique name. The name may contain any characters and can be up to 32 characters in length.

Crushing Strength - f’c

This is the crushing strength for the material as defined by most common specifications. This strength is normally defined as the stress value at a strain of 0.003 at 28 days.

Modulus of Elasticity, Ec

Select the check box if you want to override the computation of modulus of elasticity and enter your own value. If the check box is unchecked, PGSuper will compute Ec based on f'c and the density for strength values.

Density for Strength

This is the density which is used along with f’c to calculate the modulus of elasticity. If the number in this input field turns red, the density is below the minimum value for Normal Weight Concrete.

Density Used for Weight

This is the density used to determine the self-weight of a girder. The density should include the expected average effects of all reinforcement and prestressing in the structure. If the number in this input field turns red, the density is below the minimum value for Normal Weight Concrete.

Max. Aggregate Size

Enter the maximum dimension of the concrete aggregate. This value is used to check minimum reinforcement spacing requirements per LRFD 5.10.3.1.2.

K1

Enter the aggregate correction factor per LRFD 5.4.2.4.
