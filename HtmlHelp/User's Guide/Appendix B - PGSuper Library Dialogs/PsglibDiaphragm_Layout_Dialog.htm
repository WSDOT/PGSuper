Diaphragm Layout Tab

PGSuper allows you to describe the rules to define the number of intermediate diaphragms used for given span lengths. An unlimited number of rules may be specified. Each rule defines the location of one diaphragm

The loads due to intermediate diaphragms are included in the dead load analysis. The concrete for intermediate diaphragms is assumed to be the same as that used for the slab.

Using the Grid

Each row on the grid represents a rule. Use the Add, Edit and Delete buttons to manage the Rules. Double-clicking the mouse on a row will open the rule for editing.

Tip: It is wise to give the rules names that describe what the rule does for a given span range.

Editing Rules

Span Length Range

Enter the minimum and maximum span lengths for which the rule applies.

Diaphragm Dimensions

Enter the height and thickness of the rectangular intermediate diaphragms for this girder. These dimensions are used to compute the weight of the diaphragm based on the girder spacing (or internal girder void size) and the weight density of the slab concrete.

Diaphragm Type

Internal diaphragms are used to fill the interior space in U and T girders

External diaphragms exist between the outside webs of girders.

Construction

Internal diaphragms can be created either at the time of girder casting of the time of slab casting

External diaphragms may only be added at the time of slab casting

Note: The diaphragm length is measured from center to center of the exterior webs of the girders. The ends of the diaphragms are not sloped to match the shape of slope webs, like those found in U girders.

Location of Diaphragms

Diaphragms can be located using fractional measure or a fixed distance from; girder ends, span ends, or from mid-span.

Example:

Span length between 0 and 40 feet - No intermediate diaphragms

40 to 80 feet - 1 diaphragm at mid-span

80 to 120 feet - 2 diaphragms at third points

greater than 120 feet (input as 120 to 99999 feet) - 3 diaphragms at quarter points
