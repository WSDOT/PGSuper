Design Tab

Options for Compliance Checking and Automated Design

Use the check boxes to select whether PGSuper performs compliance checking (spec-checking) and/or considers the listed categories in the automated design algorithm.

“A” Dimension / Slab Offset

The “A” dimension is the vertical distance between the top of the slab and the top of the girder at the bearing line. If checking is enabled, PGSuper will report whether the slab offset is inadequate or excessive.  If design is enabled, the “A” Dimension design will attempt to find a distance that will minimize “A” while keeping the girder from impinging into the slab at the highest camber location.

If “A” dimension design is disabled, the program will use the current project value. Note that design of "A" can also be turned off in the Girder Design dialog.

More information about the "A" Dimension can be found here

Lifting

The check for lifting will report whether the girder can be lifted in the casting yard.

The lifting design algorithm will attempt to optimize values of lifting loop location, concrete strength, number of harped strands, and number of temporary strands so the girder can be safely lifted in the casting yard.

Hauling

The check for hauling reports whether the girder can be trucked to the construction site

The hauling design algorithm will attempt to optimize values of support locations, concrete strength, and number of temporary strands so the girder can be safely trucked from the casting yard to the construction site.

Strand Slope

Limiting max strand slope is a manufacturability requirement. If enabled, the algorithm will check to make sure the slope of harped strands is below the limit and will attempt to lower the harped pattern at the girder ends if it is.

Note that this option is only applicable to harped girder design.

Hold Down Force

Limiting max harped strand hold down force is a manufacturability requirement. If enabled, the algorithm will check to make sure the hold down force is below the limit and will lower the harped pattern at the girder ends if it is.

Note that this option is only applicable to harped girder design.

Anchorage Splitting and Confinement

Use this option to enable/disable the checks for splitting/bursting (5.10.10.1) and confinement (5.10.10.2) in the prestress anchorage zones.

Fill Order for Harped and Straight Strand Design

This option tells the automated design algorithm to either fill permanent strands using the global strand order defined in the Permanent Strands Tab of the girder library entry (TxDOT method), or to attempt to minimize the total number of harped strands (WSDOT method).
