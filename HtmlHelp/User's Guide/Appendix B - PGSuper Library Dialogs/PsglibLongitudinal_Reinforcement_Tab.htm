Longitudinal Reinforcement Tab

This tab allows you to define the longitudinal reinforcement layout seed data for a girder. Note that longitudinal rebar data from the library is copied to all girders when a girder type is first selected while editing the bridge model. The data can be subsequently overridden in the girder editor. Users can also re-copy the data from the library.

Select the appropriate mild steel for longitudinal rebar

Rows of Reinforcement

PGSuper allows you to describe an unlimited number of horizontal rows of longitudinal reinforcement. Rows of reinforcement are symmetrical about the girder centerline. The following data is used to describe rows of longitudinal reinforcement.

Girder Face

Rebar rows may be measured either from the top or bottom face of the girder.

Bar Size

Lists all available standard bar sizes that may be used for longitudinal rebar.

# of Bars

Specify the number of bars to be placed in the row.

Cover

Enter the distance between the specified girder face to the outside nominal diameter for the specified bar.

Spacing

Enter the center-to-center horizontal spacing between the rebars.

Note that minimum bar spacing requirements are not checked by the program. It is your responsibility to insure that bars don't overlap.

Using the Grid Control

The longitudinal reinforcement row location grid control allows you to add, modify, and delete rows of longitudinal reinforcement in a girder section. The following rules apply:

Row locations may be added above the currently-selected row by clicking the “Insert Row” button or selecting “Insert Row” from the right mouse button context menu.

Row locations may be appended the the last row of the list by clicking the “Append Row” button.

Row locations may be removed by first selecting the first column (Row #) of the row and then clicking on the “Delete Row” button. Multiple rows may be selected by holding the Ctrl or Shift keys while you select with the mouse.

All cells in the grid must be filled in. Blank cells are considered incomplete data.
