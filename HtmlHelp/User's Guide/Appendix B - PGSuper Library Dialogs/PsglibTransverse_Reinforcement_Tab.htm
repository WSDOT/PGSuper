Transverse Reinforcement Tab

This tab allows you to edit the description of transverse (shear) reinforcement seed data for a girder. Note that transverse rebar data from the library is copied to all girders when a girder type is first selected while editing the bridge model. The data can be subsequently overridden in the girder editor. Users can also re-copy the data from the library.

Mild Steel Material

Select the appropriate mild steel for the transverse rebar

Transverse Reinforcement Zones

PGSuper allows you to describe transverse reinforcement in fixed-length zones along the girder. The first zone in the list starts at the end of the girder, and the last zone in the list extends to the girder mid-span. Each zone of transverse reinforcement is described by the following:

Zone Length

Enter the length of the zone. This value must be greater than zero.

Stirrup Spacing

Enter the center-to-center spacing between the stirrups in the given zone.

Vertical Bar Size

Lists all available standard bar sizes that may be used for transverse bars with a vertical orientation.

Vertical Bars #

Number of bars (stirrup legs) crossing a shear crack in this zone.

Horizontal Bar Size

Lists all available standard bar sizes that may be used for transverse bars with a horizontal orientation.

Horizontal Bars #

Number of bars crossing a horizontal splitting plane in this zone.

Transverse Reinforcement engages deck

When checked, it is assumed the transverse reinforcement extends into the deck and is adequately developed to be considered effective in the horizontal interface shear capacity.

Top Flange Horizontal Interface Shear Steel

The top flange horizontal interface shear steel are stirrups located in the top flange of the section. These bars protrude up into the slab with adequate development length to contribute to the horizontal interface shear capacity. They are used to add strength to the slab-girder interface to resist horizontal shear forces. These stirrups are spaced evenly along the entire length of the girder. Each stirrup provides two legs of reinforcement.

Bar Size

Lists all available standard bar sizes that may be used for top flange stirrups.

Stirrup Spacing

Stirrups are spaced evenly along the entire length of the girder. Enter the center-to-center spacing.

Bottom Flange Confinement Steel

PGSuper assumes that the hoops of confinement steel placed in the bottom flange of the girder to prevent splitting are attached to stirrups. Hence, the spacing of the confinement steel is the same as the stirrup spacing. The following two values are used to specify the bottom flange confinement steel:

Bar Size

Lists all available standard bar sizes that may be used for confinement rebar.

Last Zone Containing Confinement Steel

Confinement steel will be placed from the end of the girder to the end of the specified zone. A value of none for this field indicates that no confinement steel is present.
