Girder Template Editing Dialog

The Girder Template Dialog is a multi-tabbed dialog which allows you to edit the properties of prismatic prestressed I-girders to be used in PGSuper bridge models. The Girder Template Dialog is composed of the following tabs:

Girder Dimensions

Permanent Strands

Temporary Strands

Harping Points

Longitudinal Reinforcement

Transverse Reinforcement

Intermediate Diaphragm Layout
