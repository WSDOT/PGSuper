Description Tab

This tab allows you to name your custom Project Criteria Entry and provide a description for it. You also select the version of the AASHTO LRFD Specification that this Project Criteria Entry will augment.

Project Design Criteria Entries allow you to define the version of the LRFD Specifications that you want to use in your analysis/design. They also allow you to customize certain provisions that are likely to be agency-specific, or not defined in the Specifications.

Entry Name

Each Design Criteria Entry must have a unique name. The name may contain any characters and can be up to 32 characters in length.

Description

Provide a description for the Project Criteria Entry.

Use Design Criteria Along With Provisions from the:

Select the version of the LRFD Specifications that you would like to use along with the provisions described by the current Project Criteria Entry.

Use Constants and Equations from the Following Version of the Selected Specification:

All of the LRFD Specifications are available in SI and US units. Selection of the SI version of the Specifications means that SI-specific constants will be used when evaluating specification equations.
