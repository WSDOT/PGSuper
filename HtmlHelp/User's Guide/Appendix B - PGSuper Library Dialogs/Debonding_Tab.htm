Debond Criteria Tab

Debonding Strand Limits (5.11.4.3)

These values are used to check and design debonded sections to ensure that local stress concentrations at debond locations are not excessive.

Maximum Number of Debonded Strands

Enter this value as a percentage of the total number of straight strands

Maximum Number of Debonded Strands Per Row

Enter this values as a percentage of the number of straight strands in a row

Maximum Number of Debonded Strands Per Section

These values limit the number of strands that can be debonded at a given longitudinal location. Enter both a total number and a percentage of the total number of debonded strands

Criteria for Debond Distances

Maximum Debond Length

The distance from the end of the girder to any given debond section cannot exceed this distance. This value can be entered either/or as a percentage of the girder length or a fixed distance.

Minimum Debond Length

The minimum distance between adjacent debond sections. Note that this value should always be less than the prestress transfer length for the currently selected strand material, otherwise the debond design algorithm may create designs that do not pass the specification check.

Default Debond Length

This value is used for design, and as an initial value in the debonding grid. This value cannot be less than the minimum length specified above.
