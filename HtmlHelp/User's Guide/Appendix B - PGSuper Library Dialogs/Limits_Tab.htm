Limits and Warnings Tab

Concrete Properties

Here you can specify certain limits on various concrete properties. If the limits are exceeded, warnings are posted in the Status Center. These limits do not affect specification checking or girder design.

Stirrup Zone Length and Stirrup Spacing Compatibility Warning

Select the check box if you want PGSuper to issue a warning if input stirrup spacings do not fit exactly within stirrup zones (i.e., if the number of stirrups within a zone is not an integer value). Note that a failure of this check is only a warning, not a spec check failure. The warning is issued at the top of the Spec Check report in the Spec Check Summary section.
