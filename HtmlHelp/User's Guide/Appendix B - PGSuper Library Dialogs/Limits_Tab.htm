Limits and Warnings Tab

Concrete Limits

Here you can specify certain limits on various concrete properties. If the limits are exceeded, warnings are posted in the Status Center. These limits do not affect specification checking or girder design.

General Warnings

Warn if stirrup zone length are incompatibility with stirrup spacings

Check this box if you want PGSuper to issue a warning if input stirrup spacings do not fit exactly within stirrup zones (i.e., if the number of stirrups within a zone is not an integer value). Note that a failure of this check is only a warning, not a spec check failure. The warning is issued at the top of the Spec Check report in the Spec Check Summary section.

Warn if there is a potential for girders to sag under dead load effects

Check this box if you want PGSuper to evaluate the camber immediately before of deck casting to the deflection caused by the weight of the deck and superimposed dead loads and issue a warning if the girder may sag. The girder may sag if the camber for the fast construction scenario (using minimum timing for creep) is less than the deflection due to the deck and superimposed loads.

The camber for sag detection can be based on the lower bound camber, average camber, or upper bound camber. Select the appropriate setting.
