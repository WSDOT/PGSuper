Permanent Strands Tab

This tab allows you to specify the prestressing strand grid for permanent prestressing strands.

Here are some facts about permanent strands:

  • The strand grid defines all possible permanent strand locations in a girder, and the order in which the strands are filled. Another way to think of this is a template for strand placement.  Actual strands are placed in a project by specifying the number of strands in the Strands Tab of the Girder Editing dialog.

  • There are two basic types of permanent strands.

    1. Straight Permanent Strands can be placed anywhere within a girder section and are straight along the entire girder.

    2. Web Strands are permanent strands that can only be placed within the thinnest portion of a girder section's web(s). Web Strand templates have the unique capability to be vertically adjusted within the web region. There are two types of Web Strands: Harped Strands and Straight-Web strands. Harped strands are draped at the girders harping points, and Straight-Web strands are straight along the entire length of the girder.

  • Permanent strands are placed and tensioned at casting time and remain in the girder for its lifetime.

  • Harped and Straight-Web strands are bonded along the entire length of the girder.

  • Individual straight strands can be debonded if the allow debonding option is selected.

Strand Grid

Use the strand grid control to define a template of possible strand locations and fill order of permanent straight and web strands in the girder.

The strand location grid control allows you to add, modify, and delete template locations for strands in a girder section. Double-click on the grid row to edit the strand type and location. The following rules apply:

  • The origin of the coordinate system for strand locations is the bottom center of the section as shown in the Figure above. All PGSuper sections use this convention.

  • X values cannot be negative.

  • If a non-zero X value is given, two strands are added for the fill location: one strand at (X,Y) and another at (-X,Y). This enforces the requirement that all strand templates must be symmetric about the Y axis.

  • If the X value of a strand coordinate is zero, a single strand will be placed at that location.

  • Strands are added to the section in the given Fill # sequence.

  • Strand locations may be added above the currently-selected row by clicking the “Insert” button.

  • Strand locations may be appended the last row of the list by clicking the “Append” button.

  • Strand locations may be removed by first selecting the row and then clicking on the “Delete Row” button.

  • Strand locations may be edited by double clicking on a row or by first selecting the row and then clicking on the “Edit” button.

  • Strand locations may be moved in the fill sequence by selecting the row and then clicking on the "Move Up" or "Move Down" buttons.

  • The fill sequence for harped strands can be reversed by clicking on the "Reverse Harp Strand Sequence" button.

  • The position of strands can be generated by clicking on the "Generate Strand Positions" button.

  • Straight Permanent strands must be located inside of the girder.

  • Web strands must lie within the thinnest portion of the girder's web(s), and within the specified allowable vertical offsets.

  • Allowable vertical locations of web strands can be controlled by the prescribed adjustment limits.

Use Different Harped Locations at Girder Ends

This option allows you to define different locations of a harped strand at the ends of the girder and at the harping points. This option is most often used to describe the harped bundles at harping points used by WSDOT I girders.

Note that you do not need to use this option if the relative distance between strands is the same along the entire length of the girder. In this case harping can be achieved by adjusting the vertical offset of strand patterns at the girder ends and harping points.

Debonding of Straight Strands

The Edit Strand Location Dialog allows selection of whether individual permanent straight strands are allowed to be debonded or not.

When shown in the permanent strand grid, strands of type "Straight" may not be debonded. Strands of type "Straight-DB" can be debonded by the design algorithm or in the girder editor dialog.

Strand Options

Coerce Odd Number of Web Strands

When selected, PGSuper will force (coerce) the lowest (first in the fill order) pair of web strands to alternate between a single strand at X=0.0, and two strands at the prescribed +/- X values. This allows a strand grid that contains only pairs of coordinates to place strands one at a time.

Note that all end strands in the web strand grid must have positive X values (i.e., filled in pairs) when this feature is enabled.

This feature is used on some WSDOT I-Beams or Ribbed Girders with an odd number of webs.

Vertical Adjustment of Web Strand Grid

If enabled, this option allows harped and straight-web strand templates to be vertically offset. Harped strands can be offset differently at the girder ends or at harping points, thus forming the harped drape. Straight web strands use a single offset along the entire girder length.

Design Increment

In general, web strands can be adjusted continuously up and down within their adjustment limits through user-input adjustment values.

The design increment defines the step size to be used by the automated design algorithm when adjusting strand offsets. The increment must be less than or equal to the maximum adjustment value.

If this value is zero, the design algorithm will make continuous adjustments within the offset limits.

Adjustment Limits

These values define the range of possible uppermost and lowermost web strand Y locations. Web strands may not lie above or below these limits. Limits can be measured downward from the top of the girder or upward from the bottom.
