Vehicular Live Loads

Use this dialog to define a Vehicular Live Load.

Apply Using

Select how the live load is applied

Truck Only

This loading consists only of a truck

Lane Load Only

This loading consists only of a lane load

Sum of Lane Load and Truck

The response of the lane load and truck are added together

Envelope of Lane Load and Truck

The response of the lane load and truck are enveloped

Lane Load

Input the magnitude of the lane load

Truck Configuration

Describe the truck configuration. Trucks are defined as a series of axle weights and spacing. Use the Add and Delete buttons to define the axles

Select the check box if only axles contributing the to extreme force effect under consideration are to be considered.
