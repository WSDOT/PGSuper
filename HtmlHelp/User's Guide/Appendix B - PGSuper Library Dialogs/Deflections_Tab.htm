Deflections Tab

Optional Live Load Deflection Criteria

Select whether to check LRFD Provision 2.5.2.6.2 for deflection due to live loads. The default deflection ratio is L/800.

The LRFD optional deflection live load is specified in 3.5.1.3.1. Note that the live load is applied to the entire bridge structure using the stiffness of the slab, girders and barriers. The barrier concrete material is assumed to be the same as the slab concrete. Slab fillet and slab offset ("A" dimension) effects are ignored (the deck is placed directly atop the girders).
