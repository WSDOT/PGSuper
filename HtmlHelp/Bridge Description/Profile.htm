Alignment Description - Profile

Profile

The roadway profile is defined by a series of grade breaks (PVI) and vertical curves.

Straight Grade

Enter a Station where the roadway centerline elevation and grade are known. Enter the Elevation and Grade. Leave the Curve Table empty.

Vertical Curves and PVIs

Describe the entry grade of the first vertical curve by supplying data for the Station, Elevation, and Grade fields. Describe each vertical curve with its PVI Station, Exit Grade, the distance from the beginning of the curve to the PVI (L1) and the distance from the PVI to the end of the curve (L2).

The exit grade of the previous curve is taken to be the entry grade of the next curve.

The total length of the vertical curve is L1 + L2.

For symmetrical vertical curves, enter the total curve length as L1 and leave L2 blank.

Leave L1 and L2 blank to define a point of vertical intersection (PVI) for a grade break.

NOTE: While grade breaks do occur, they are not common. When a grade break is modeled,PGSuper will put a warning in the Status Center telling you that a zero-length vertical curve has been modeled. If you intended to model a grade break then no action is necessary. If you meant to model a vertical curve, then hopefully this status item will alert you to the input error.
