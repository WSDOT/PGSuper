Bridge Description - Superstructure

The superstructure for beam and slab bridges is fairly simple to describe. On the Superstructure tab, you describe the layout of the girders and the slab cross section.

Superstructure Components

Describe the components of the superstructure.

Girder Type

Use the drop down list to select a type of girder.

Girder Name

Use the drop down list to select a girder from the library.

Tip: You can create new girder shapes by creating new girder entries in the library.

Girder Orientation

Select a girder orientation from the drop down list. Girders may be oriented in the cross section as Plumb (their vertical axis in the direction of gravity) or Normal to the deck at the start bearing, mid-span, or end bearing.

Typically I-beams are oriented Plumb and deck bulb tees, slabs, and U-beams are oriented normal to the deck.

Transverse Connectivity of Adjacent Girders

For girders that are always adjacent to one another, such as box beams or deck bulb tees, the live load distribution factors are a function of the transverse connectivity. Select the type of transverse connectivity. The Superstructure Description will tell you the type of cross section PGSuper will consider when computing distribution factors.

Deck Type

Use the drop down list to select a deck type. The available deck types depend on the girder type selected.

Left/Right Railing System

Use the drop down list to select a railing system from the traffic barrier library.

Tip: You can create new traffic barrier types by creating new traffic barriers entries in the library.

Superstructure Layout

Describe framing plan of the superstructure.

Number of Girder Lines

Using the up and down arrows, enter the number of girder lines. In this version of PGSuper, all girders are assumed to be parallel.

Girder Spacing

Enter the spacing of the girders. The girder spacing is measured normal to the alignment at the centerline of bearing. Some girder types require the girders to be placed adjacent to one another. In this case, the girder spacing input will be disabled.
