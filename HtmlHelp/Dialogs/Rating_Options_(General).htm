Rating Options (General)

Load Rating Criteria

Use the drop down list to select a load rating criteria.

System Factor

Input system factors for flexure and shear. See MBE 6A.4.2.4 for guidance.

Average Daily Truck Traffic

Enter the average daily truck traffic (ADTT) for one direction. If the ADTT is unknown, input the keyword "Unknown".

Include pedestrian live load

Check this box if pedestrian live load is to be included in the load rating analysis. See MBE 6A.2.3.4 for guidance.

Rating Type

Select the type of load rating to be preformed by check the appropriate boxes.
