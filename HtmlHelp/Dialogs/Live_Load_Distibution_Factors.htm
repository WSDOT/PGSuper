Live Load Distribution Factors

This project-specific dialog allows you to chose whether live load distribution factors are to be computed using the method specified in the current Project Criteria library entry, input manually, or computed using only the lever rule. Options are also available to override safeguards when parameters lie outside of the ranges of applicability specified in the AASHTO LRFD.

Compute Live Load Distribution Factors in Accordance With the Currently Selected Project Criteria

When this option is selected, distribution factors are computed using the methods specified on the Bridge Site 3 tab of the currently selected Project Criteria Library entry. The Ranges Of Applicability (ROA) for use of the distribution factor equations, are girder-type dependent and defined in section 4.6.2.2 of the LRFD Specifications. By default, PGSuper will refuse to analyze a structure where any range of applicability rule is broken. However, there may be cases when you want to override this behavior, and options are given below:

When a Parameter is Outside of the ROA:

Halt the Analysis And Issue a Warning

This is the default behavior. PGSuper will not complete the analysis unless you override this.

Ignore the ROA Requirements and Perform the Analysis Using the Equations

The LRFD equations will be used for computation of distribution factors regardless of their applicability.

Note: Ignoring the range of applicability is generally NOT RECOMMENDED, and should only be selected for special structures where you need to control the computation of distribution factors. Checks using hand calculations, or other methods, should always be done to insure that the factors are computed as you intended.

Use the Lever Rule Only for Cases Outside of the ROA Requirements

The LRFD equations will be used for computation of distribution factors where parameters are inside of the Range of Applicability. However, the lever rule will be used for cases outside the ROA for the equation. This option may seem safer than the previous option, however, the same amount of caution should be applied when using it.

Compute All Live Load Distribution Factors Using the Lever Rule

When this option is selected, all distribution factors are computed using the Lever Rule. Note that the LRFD skew factor equations are still applied, however, any ROA requirements for the skew equations are ignored. Again, use caution when using this option.

Directly Input All Live Load Distribution Factors

When this option is selected, user-input distribution factors are used directly.

Tip: Use this dialog to enter your own live load distribution factors if you don't like the values PGSuper calculates.
