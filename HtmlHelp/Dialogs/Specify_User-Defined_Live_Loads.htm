Live Loads

Live loads are defined for two different load categories which are enveloped to determine responses for the following limit states:

1) Design Limit States

  • Service I

  • Service IA

  • Service III

  • Strength I

2) Permit Limit State

  • Strength II

This dialog is used to select the live loads whose responses will be enveloped for each category. Use the check marks in each list box to select the Vehicular Live Loads whose response you wish to add to the live load response envelope for that category. Vehicular Live Loads are defined in the PGSuper Library. PGSuper pre-defines the LRFD HL-93 live load.

Impact Factors

Truck and Lane load dynamic load allowance factors (impact factors) may be entered as well.

Application of Pedestrian Live Loads on Sidewalks when Sidewalk is Present

If a sidewalk of adequate width is present, pedestrian live loads are applied to those girders, mating surfaces, or webs (GMSW’s) loaded by the sidewalk dead load. The pedestrian live load is then combined with the vehicular live loads for the selected limit state. The possible options are:

1) Apply only vehicular live loads (ignore pedestrian loads all together)

2) Envelope pedestrian live loads with vehicular live loads

For this option, vehicular live loads are applied to all girders and enveloped with the application of pedestrian live loads for applicable GMSW’s. A multiple presence factor, m=1.0 is used for the pedestrian live load.

3) Apply pedestrian live loads concurrently with vehicular live loads

For this option, vehicular live loads are applied to all girders and summed with the application of pedestrian live loads for applicable GMSW’s

Notes:

  • If you do not select any live loads for the Strength II category, then Strength II responses will not be computed.

  • The Optional deflection vehicle as defined by 3.6.1.3.2 is a part of the HL-93 response envelope. You must select HL-93 in order to activate the deflection vehicle.

  • Refer to Chapter 5 of the PGSuper User's Guide for more information about load generation.

  • Refer to other notes in the Vehicular Live Loads section of the Theoretical Manual

  • Pedestrian loads will not be applied if the sidewalk width is less than the minimum width specified on the Project Criteria Bridge Site 3 tab.

  • When pedestrian live loads are applied, dead loads from the sidewalk and interior barriers are applied as if traffic is allowed to mount the sidewalk.
