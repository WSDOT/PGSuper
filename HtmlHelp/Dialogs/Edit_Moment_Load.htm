Edit Moment Load

This dialog is used to edit user-defined moment loads in the model.

Load Case

Select the Load Case that the load is to be applied to. The available options are DC, DW and LL+IM as defined in the LRFD Specifications. LL+IM can only be applied in Bridge Site Stage 3.

Stage

Select the stage when the load is first applied. In PGSuper all user-defined loads are permanent and their results are accumulated through all subsequent stages.

Component(s)

Select the girder that the load is to be applied to. You can select "All Spans" or "All girders" to apply the load to more than one girder.

Location

Select the location along the girder where the load is to be applied. The load can be applied at the start or end of the span.

Magnitude

Enter the magnitude of the load. Positive moments are counter clockwise
