Report Options

Use this dialog to select display options for one of the many reports PGSuper has to offer.

Select Girder

Depending on the type of report, use the drop down lists or the multi-girder select grid to select a span(s) and/or girder(s) for which the report will be created.

Select chapters to be included in the report

Scroll through the list of chapters and place a check mark next to the ones you want included in the report. You can select several chapters at a time using the standard Ctrl+Click and Shift+Click mouse operations.
