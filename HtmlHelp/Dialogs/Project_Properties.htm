Project Properties

When creating a new PGSuper project, one of the first things that you will do is supply some basic information about the project. These properties are used to describe a bridge project and to identify the engineer performing the work.

Bridge

Enter the name of the bridge

Bridge ID

Enter a unique identifier for the structure

Job Number

Enter a job number or other accounting related information

Engineer

Enter your name.

Company

Enter your company name

Comments

Enter any comments that you would like to save with the PGSuper project. 

Note: The Engineer and Company fields are automatically filed when you create a new project file.
