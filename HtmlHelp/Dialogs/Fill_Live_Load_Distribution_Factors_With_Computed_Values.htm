Fill Live Load Distribution Factor Grid With Computed Values

This dialog allows you to automatically fill selected entries in the user-input live load distribution factors grid with computed values. Data in the input grid data will be replaced by computed or input values for the selected girder(s) or Pier(s).

Note that this is a one-time operation. Values will NOT be automatically updated if the bridge is later modified.

If computation fails for any girder (probably due to range of applicability reasons), an error message will be posted and grid data remains unchanged.

Live Load Distribution Factor Computation Method

Select the method to compute the factors to be written into the user-input grid. The options for computation methods and Range of Applicability are the same as those on the Live Load Distribution Factors dialog. However, the data will only be filled once and then you can change any data in the grid to suit your needs.

Fill Live Load Distribution Factors With a Single Value

This option allows a single floating point value to be assigned to all selected girders and piers in the grid.

Selection of Girders and Piers

Use the combo boxes to select which girders and/or piers that the new data will be copied to.
