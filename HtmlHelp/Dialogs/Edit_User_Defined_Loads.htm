Edit User Defined Loads

This view provides a convenient place to view, add, delete, or edit all user-defined loads in the model. All currently defined loads are shown in the list with a short description showing the magnitude of the load, its location, what stage it is applied in, and which load case it is applied to. Loads can be edited by double clicking on the description.

Edit Load

Pressing this button will bring up the editing dialog for the currently selected load. If no load is selected, this button is disabled.

Delete Load

Pressing this button will delete the currently selected load. If no load is selected, this button is disabled.
