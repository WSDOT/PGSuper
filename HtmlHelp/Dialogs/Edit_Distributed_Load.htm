Edit Distributed Load

This dialog is used to edit user-defined distributed loads in the model.

Load Case

Select the Load Case that the load is to be applied in. The available options are DC, DW and LL+IM as defined in the LRFD Specifications. LL+IM can only be applied in Bridge Site Stage 3.

Stage

Select the stage when the load is first applied. In PGSuper all user-defined loads are permanent and their results are accumulated through all subsequent stages.

Component(s)

Select the girder that the load is to be applied to. You can select "All Spans" or "All girders" to apply the load to more than one girder.

Load Type

Select Trapezoidal or "Uniform along entire Span"

Magnitude

Enter the magnitude of the distributed load at the start and end location of the load. Positive loads are applied in the direction of gravity.

Fractional

If this option is selected, loads will be located from the left support using a fraction of the flexible span length. The Location value may vary between 0.0 and 1.0.

Location

Only used for Trapezoidal loads. Select the location along the girder, measured from the left support, of the left and right ends of the load. The left end location must be smaller than the right end location. The entire load must be applied within the flexible span length of the girder. If it is not, the load will be ignored in the analysis and a message stating so will be placed in the Status Center.
