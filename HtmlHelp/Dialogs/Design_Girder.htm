Girder Designer

The Girder Designer feature in PGSuper is used to "take an educated guess" at an initial design for a selected girder in your bridge. With this dialog, you can have PGSuper attempt to design a girder for flexure and/or shear.

Select Girder(s)

Two options are available for selecting girders:

  1. Design a single girder: Use the drop down lists for the span and girder to select the girder that will be designed.

  2. Design multiple girders: Click the button to bring up the multiple girder select dialog and select the desired girders on the multi-girder select grid.

Perform Design For Flexure

When checked, PGSuper attempts to design the girder for flexure

Retain Current "A" Dimension (Slab Offset)

If the Slab Offset is considered in design (as set in the Project Criteria library), PGSuper will attempt to find an optimal offset for the selected girder. However, engineers will typically want to keep the slab offset constant when computing a final design. Turn this option on if you do not want the design algorithm to design the slab offset for the current girder.

Note: A-Dimension design is not available if there is no slab, if slab offset design is disabled in the Project Criteria library, or if multiple girders are selected for design. Hence, this control will be disabled for these cases.

Perform Design For Shear

When checked, PGSuper attempts to design the girder for shear

Tip: If your PGSuper Administrator has defined standard stirrup configurations for your girders, as is done at WSDOT, you generally won't want PGSuper to design for shear.

Remarks

Refer to the chapter on design in the User's Guide for more information on designing girders.
