Span Details - Girders

Girder spacing are defined at each end of the span.

Number of Girders

Enter the number of girders in this span. If the number of girders is the same for all spans, click on the hyperlink text to enable span-by-span girder definition. If girders are defined span-by-span, clicking on the hyperlink text will cause all spans to have the number of girders specified for this side of the pier.

Slab Offset

Enter the slab offset for all girders in this span. If a single slab offset is used for the entire bridge, or if the slab offset is defined girder-by-girder,  click on the hyperlink text to enable span-by-span slab offset definition. If the slab offset is defined span-by-span, clicking on the hyperlink text will apply the current value to all the girders in this span or to the entire bridge depending on the slab offset mode when this dialog was opened.

Girder Name

Use the grouping features of the girder name grid to create groups of girders. Enter the girder name in the grid.

The grid can be expanded to provide a column for each girder by selecting the column heading, right clicking, and choosing the Expand command

Select columns to join together, right click and choose the Join command

Girder Spacing

Select the girder spacing datum from the drop down list. Refer to the Defining Girder Spacing section in the Theoretical Manual for additional details.

Use the grouping features of the spacing grid to create groups of girder spacing. Enter the girder spacing in the grid. Girder spacing is grouped the same way girder names are grouped.

The grid can be expanded by selecting the column heading, right clicking, and choosing the Expand command

Select columns to join together, right click and choose the Join command
