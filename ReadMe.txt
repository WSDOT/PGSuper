2.0 Build Instructions
----------------------

2.1 Compilers
The following compilers and development environments were used to
create this software:
Microsoft Visual C++ 2008
Microsoft Visual Basic 6

2.2 Libraries
You will need to build or install the following libraries on your computer:
WBFL
Microsoft XML Parser (distributed with IE5) [Note: We would perfer to use
an Open Source Automation Compatible XML parser]

None specified
