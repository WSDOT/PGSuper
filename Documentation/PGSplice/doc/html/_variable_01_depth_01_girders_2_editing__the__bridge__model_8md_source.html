1 Editing the Bridge Model {#tutorial_variable_editing_the_bridge_model}
2 ==============================
3 The bridge modeled in the 2 Span template isn't exactly what we need. Let's learn a couple new editing techiques and make this bridge model fit our example.
4 
5 In Place Editing
6 ------------------
7 Another unique element of the _Bridge-centric_ user interface is in place editing. We want span 2 to be 150 ft in length, so simply click on the span length (180 ft) and type in 150 ft. Press Enter and the span length is changed.
8 
9 ![](Tutorial_Variable_Inplace_Editing.png)

11 Creating an Erection Tower and Closure Joint
12 --------------------------------------------
13 The default bridge has two segments per girder and we need three. We also need an erection tower at Station 1+25. We will create the erection tower and use a Closure Joint connection which will break the first segment into two pieces.
14 
15 1. Select *Edit > Insert Temporary Support*
16 2. On the General tab, change the station 1+75 ![](Tutorial_Variable_TS_General.png)
17 3. On the Connection tab, change the Segment Connection to Closure Joint and set the Installation Event to Event 7: Cast Closures and Deck ![](Tutorial_Variable_TS_Connection.png)
18 4. Press [OK]
19 
20 In the Bridge View, you'll see that we've added an erection tower and there are three segments per girder.
21 
22 Selecting the Girder Type
23 --------------------------
24 The default girder type is a WF95PTG girder. This is a constant depth girder. We need to select a different girder type.
25 
26 1. Select *Edit > Bridge*
27 2. On the General tab, change the girder type to WF-PTG.
28 3. Press [OK]
29 
30 Now that the basic layout of the bridge and girder configuration is complete, we'll define the details of the variable depth pier segment.
