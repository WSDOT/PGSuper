1 Profile {#ug_dialogs_profile}
2 ==============================================
3 The roadway profile is defined by a series of grade breaks (PVI) and vertical curves.
4 
5 The profile begins with a reference point before the start of the bridge and then is defined by a sequence of vertical curves.
6 
7 The entry grade of the first vertical curve and the exit grade of the last vertical curve are projected backwards and forwards along the profile.
8 
9 Item | Description
10 -----|------------
11 Station | Station of the reference point
12 Elevation | Profile elevation at the reference point
13 Grade | Profile grade at the reference point in the direction of increasing station
14 Vertical Curves | The vertical curves for this profile
15 [Add] | Adds a vertical curve
16 [Remove] | Deletes the selected vertical curves
17 [Sort] | Sorts the vertical curves based on PVI Station
18 
19 Vertical curves are defined by the following parameters:
20 Item | Description
21 -----|-------------
22 PVI Station | Point of Vertical Intersection station
23 Exit Grade | The profile grade at the exit point of the vertical curve
24 L1 | Distance from the beginning of the vertical curve to the PVI, or the total length of the vertical curve if L2 is left blank
25 L2 | Distance from the PVI to the end of the vertical curve.
26 
27 > NOTE: The PVI of the first vertical curve can be used as the Reference Point
28 
29 > NOTE: The exit grade of the previous curve is taken to be the entry grade of the next curve.
30 
31 > NOTE: The total length of the vertical curve is L1 + L2.
32 
33 > NOTE: For symmetrical vertical curves, enter the total curve length as L1 and leave L2 blank.
34 
35 > NOTE: Leave L1 and L2 blank to define a point of vertical intersection (PVI) for a grade break.
36 
37 > NOTE: While grade breaks do occur, they are not common. When a grade break is modeled, a warning will be put in the Status Center telling you that a zero-length vertical curve has been modeled. If you intended to model a grade break then no action is necessary. If you meant to model a vertical curve, then this status item will alert you to the input error.
