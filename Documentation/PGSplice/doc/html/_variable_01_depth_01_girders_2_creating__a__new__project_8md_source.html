1 Create a PGSplice Project {#tutorial_variable_new_project}
2 ==============================
3 New PGSplice projects are created from project templates. The templates contain standard configurations for bridge geometry and components, appurtenances, loading, and design criteria. Since this is a two span I-Girder bridge, the easiest place to start is with the 2 Span I-Girder template.
4 
5 > TIP: You can create your own project templates and customize their organization. See @ref ug_configurations in the @ref user_guide for more information.
6 
7 
8 Start BridgeLink
9 --------------
10 To start the BridgeLink application framework, select *Start > Programs > Engineering Applications > BridgeLink*.
11 
12 
13 Create a New Project
14 ---------------------
15 1. Select *File > New*. This brings up the New Project window. ![](Tutorial_Variable_NewProject.png)
16 2. Select PGSplice Project Templates and then I-Beams from the Project types list.
17 3. Select 2 Span from the template list. Press [OK].
18 4. This will create a new project and bring up the Project Properties dialog where you can enter information about your project. ![](Tutorial_Variable_Project_Properties.png)
19 
20 > NOTE: The Project Properties dialog may not open when creating a new project. If this is the case, select *File > Project Properties* or *Project > Properties*. Use the check box at the bottom of the dialog to control whether the dialog opens when new projects are created.
21 
22 Finally the default configuration for this type of bridge will be displayed in the Bridge Model View.
