1 General {#ug_dialogs_girder_details_general}
2 ==============================================
3 Define the details of a spliced girder.

5 Segments
6 ----------
7 Click on the combo box to select if using the same segment type for each girder individually or a single segment type for the entire bridge.
8 
9 When the segment type is defined for each girder individually, select the segment type from the drop down list. The segment types in the list are taken from the Girder library.
10 
11 The segment grid lists the segments and closure joints that make up the girder. Press [Edit] to edit the details of these components.

14 Ducts/Tendons
15 --------------
16 Define the geometry of the post-tensioning ducts and the tendons
17 
18 Item | Description
19 -----|-------------
20 Duct Grid | Grid listing the general parameters of the ducts and tendons. See Duct Grid below for details.
21 Duct Material | Select the duct material. Used for evaluation the provisions of LRFD 5.4.6.1
22 Strand Size | Select the tendon strand size
23 Installation Method | Select the strand installation method. Used to evaluate the provisions of LRFD 5.4.6.2
24 
25 ### Duct Grid ###
26 Enter the following information in the duct grid. The first two items define the duct geometry. The remainder of the items define the tendon and the jacking force parameters.
27 
28 Item | Description
29 -----|--------------
30 Type | Select a duct type from the drop down list. The duct types are defined in the Ducts library
31 Shape | Select the shape of the duct. Press [Edit] to edit the details of the duct geometry.
32 # Strands | Enter the number of strands in the tendon
33 Pjack | Check the box to enter a jacking force, otherwise the maximum jacking force is used
34 Jacking End | Select the live end for tendon jacking.
35 Event | Select the event when the tendon is stressed
36 
37 
38 Slab Offsets
39 ---------------
40 Click on the combo box to toggle between defining the slab offset for each girder individually or a single slab offset for the entire bridge.
41 
42 When the slab offset is defined for each girder individually, enter the slab offset dimension for each pier supporting this girder.
43 
44 Condition Factor
45 -----------------
46 Select the condition of the girder from the drop down list. If "Other" is selected, enter the condition factor.
