1 Layout {#ug_dialogs_pier_description_layout}
2 ==============================================
3 Select the pier model type.
4 
5 An idealized pier model is defined by a simple idealized boundary condition as defined on the @ref ug_dialogs_pier_description_connections page.
6 
7 A physical pier model is defined by the physical attributes of the pier including its material properties, cross beam, and column arrangement.
8 
9 Material Properties
10 -------------------
11 Define the material properties of the cross beam and columns.
12 
13 Item | Description
14 -----|--------------
15 f'c | Concrete strength
16 Ec | When checked, enter the modulus of elasticity of the concrete. Otherwise, the modulus is computed from the AASHTO equation
17 [More Properties...] | Edit additional concrete properties. Pre-defined concrete can be retrieved from the library using this option.
18 
19 Pier Cap Dimensions
20 --------------------
21 Define the dimensions of the pier cab or cross beam.
22 
23 Enter the dimensions is the various fields as defined by the schematic show on this tab.
24 
25 Column Layout
26 --------------
27 Define the layout of the columns.
28 
29 Item | Description
30 -----|-------------
31 Base of column | Define how the base of column is located.
32 Longit. Base Fixity | Define the fixity of the base of the column for longitudinal plane frame analysis.
33 Column Grid | Define the columns (see below for more information)
34 Locate the columns | Use these controls to transversely locate the columns (see below for more information)
35 
36 ### Column Grid ###
37 Define the columns in this grid
38 
39 Item | Description
40 -----|-----------
41 Height | Height of the column
42 Bottom Elevation | Bottom of column elevation
43 Transv. Base Fixity | Define the fixity of the base of the column for transverse plane frame analysis. This analysis is not performed by PGSuper, however this information is used by the XBRate cross beam load rating application
44 Shape | Select the cross sectional shape of the column
45 Diameter/Width | Enter the diameter of a circular column or the width (dimension in the plane of the pier) of a rectangular column
46 Depth | Enter the depth (dimension normal to the plane of the pier) of a rectangular column
47 S | Enter the center to center column spacing
48 
49 ### Location of Columns ###
50 The columns must be located transversely in the plane of the pier.
51 
52 Item | Description
53 -----|----------
54 Column | select a column to locate.
55 Offset | Enter the offset of the selected column from the selected datum
56 Datum | Select the datum from where the column location is measured.
