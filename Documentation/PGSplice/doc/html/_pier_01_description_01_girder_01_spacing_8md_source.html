1 Girder Spacing {#ug_dialogs_pier_description_girder_spacing}
2 ==============================================
3 Define the girder spacing.
4 
5 Click on the combobox to select between defining parameters for the entire bridge and for just this span.
6 
7 Item | Description
8 -----|--------------
9 Number of Girders | Enter the number of girders
10 Girder Spacing | Use the drop down list to select how and where the girder spacing is measured
11 Girder Spacing Grid | Enter the spacing between girder lines
12 Locate | Use these parameters to transversely locate the girders
13 [Copy to End/Start of Span] | Press to copy the girder spacing information to the opposite end of the span
14

29 Transversely Locating Girders
30 ------------------------------
31 The girders must be located transversely with respect to the alignment. The location of the girder is measured in the same direction, and along the same line, as the girder spacing.
32 
33 Item | Description
34 -----|-------------
35 Locate Drop Down List | Select a specific girder, or "Center of Girders", to locate transversely. "Center of Girders" is the mid-point between the centerline of the left and right exterior girders
36 Location | Enter the location of the selected girder
37 From Drop Down List | Select the location from which the selected girder is measured.
