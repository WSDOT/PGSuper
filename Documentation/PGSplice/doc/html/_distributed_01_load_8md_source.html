1 Distributed Load {#ug_dialogs_distributed_load}
2 ==============================================
3 Add or edit a user defined distributed load. Distributed loads can be uniform over the full length of a span or trapezoidal over a portion of the span.
4 
5 Item | Description
6 -----|--------------------
7 Load Case | Select the Load Case that the load is to be applied in. The available options are DC, DW and LL+IM as defined in the LRFD Specifications. LL+IM can only be applied in Event 5: Final with Live Load (Bridge Site 3).
8 Event | Select the construction event when the load is first applied. In PGSuper all user-defined loads are permanent and their results are accumulated through all subsequent stages.
9 Span/Girder | Select the span and girder that the load is to be applied to. You can select "All Spans" or "All girders" to apply the load to more than one girder.
10 Load Type | Select Trapezoidal or "Uniform along entire Span"
11 Magnitude | Enter the magnitude of the distributed load at the start and end location of the load. Positive loads are applied in the direction of gravity.
12 Location | Only used for Trapezoidal loads. Select the location along the girder, measured from the left support, of the left and right ends of the load. The left end location must be smaller than the right end location. The entire load must be applied within the flexible span length of the girder. If it is not, the load will be ignored in the analysis and a message stating so will be placed in the Status Center. If the Fractional Measure option is checked, the load will be located from the left support using a fraction of the span length.
13 Description | Enter a description for this load
