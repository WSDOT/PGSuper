1 Apply Loads {#ug_dialogs_apply_loads}
2 ==============================================
3 Define loads that are applied during a construction event.
4 
5 Select the loads applied during this event.
6 For common loads, check the box next to the loading condition to apply the load during this construction event. If the box is unchecked, the loading is not applied to the structure.
7 
8 For user defined loads, the loads in the right hand list are applied during this construction event.
9 
10 > NOTE: If a box is filled with a square the load is applied to the structure in a different event. Check the box to move the load to this event.
11 
12 Item | Description
13 -----|---------------
14 Traffic Barrier/Railing System | Check to construct the traffic barrier in this event
15 Overlay | Check to install the overlay in this event.
16 Open to Traffic | Check to apply the design live load. The design live load will be applied in the analysis interval corresponding to this event, and all analysis intervals thereafter.
17 Load Rating | Check to indicate that the rating live load is applied to the analysis interval corresponding to this event.
18 User Defined Loads to be Applied | List of user defined loads applied in other events or not applied to the structure.
19 User defined loads applied during this event | List of user defined loads applied during this event
20 [>] and [<] | Move loads between the two lists
