Live Load Distribution Factors (TxDOT){#tg_lldf_txdot}
======================================
TxDOT deviates from the AASHTO LRFD Specifications for several girder types. A description of the deviations can be found in the TxDOT Bridge Design Manual - LRFD. 
Go to http://onlinemanuals.txdot.gov/txdotmanuals/lrf/index.htm for details.
