Introduction

PGSuper (Prestressed Girder SUPERstructures) is Precast-Prestressed Girder design, analysis, and load rating software for the AASHTO LRFD Bridge Design Specifications. PGSuper is part of the BridgeLink Suite of Engineering Tools. This software tool is jointly developed by the Washington and Texas Departments of Transportation and is a member of the Alternate Route Project.
