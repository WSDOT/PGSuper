1 Girder View {#ug_girder_modeling_girder_view}
2 ==============================================
3 The Girder View provides a graphical representation of a girder. The view is divided into three regions, the Control Bar, the Elevation View, and the Section View.
4 
5 ![](GirderView.png)
6 
7 Control Bar
8 ------------
9 The Control Bar is located at the top of the view. The Control Bar is used to control the contents of the Girder View.
10 
11 ### Selecting a Girder ###
12 To select a girder to view:
13 1. Select the girder using the drop down lists
14 2. Check the Synchronize with Bridge View selection to keep the same girder selected in both views.
15 
16 ### Selecting a Construction Event ###
17 Use the drop down list to select the construction event for the view. The girder and applied user defined loads are shown for this event.
18 
19 User Defined Loads
20 ------------------
21 User defined loads can be managed in the Girder View. If user defined loads are present on the selected girder and applied in the select Construction Event, they will be displayed graphically in the elevation view.
22 
23 To create a new load, drag the point load, distributed load, or moment load icon from the control bar onto the girder at the location near where you want to apply the load. Once the load is dropped onto the girder, its editing window will be opened so you can provide specific information about the load.
24 
25 Loads may then be edited by right-clicking on them an selecting *Edit Load* from the context menu or by simply double-clicking on them.
26 
27 The Legend helps define the load cases that user defined loads are applied in. You can move the legend around the view by dragging it and you can change the number of rows shown in the legend by right-clicking on it.
28 
29 > NOTE: Showing user defined loads and the legend can be toggled on and off. If these items are not being displayed, the view settings probably need to be enabled. See @ref ug_dialogs_girder_view_settings for more information.
30 
31 > NOTE: See @ref ug_loading in the @ref user_guide for more information about User Defined Loads.
32 
33 Elevation View
34 ---------------
35 The Elevation View shows the a representation of the girder elevation. This view shows the bearing points, span length, girder length, location of harping points, prestressing strands, longitudinal reinforcing steel, and the C.G. of the prestressing force.
36 
37 ### Selecting a Cross Section ###
38 There are several methods for selecting a cross section in the Girder View.
39 
40 Use any of the following to select a cross section:
41 1. Press [Section Cut] on the control bar
42 2. Double click on the Section Cut icon
43 3. Drag and drop the Section Cut icon.
44 
45 Options 1 and 2 opens the @ref ug_dialogs_girder_view_section_cut window. Using the slider, select a location for the section cut.
46 
47 
48 Section View
49 -----------------
50 The Section View displays the girder cross section at the section cut location. The section can include the location of the prestressing strands and
51 longitudinal reinforcing steel, the C.G. of the prestressing force, and the height and width dimensions of the girder.
52 
53 Context Editing
54 ----------------
55 You can quickly edit any of the girder attributes by selecting an attribute category from the context menu. The context editing commands will take you directly to the appropriate location in the Girder Details window.
56 
57 Other options are available from the context menu as well, including Girder Design, creating user defined loads, selecting a cross section, and reporting.
58 
59 > TIP: To quickly edit the girder, double click anywhere in the view.
60 
61 Customizing the Girder View
62 ---------------------------------
63 The Girder View can be customized by enabling and disabling the various options.
64 
65 To customize the view select *View > View Settings > Girder View*. This will open the View Settings window. Customize the elevation and section views by enabling and disabling the various options on their respective tabs.
