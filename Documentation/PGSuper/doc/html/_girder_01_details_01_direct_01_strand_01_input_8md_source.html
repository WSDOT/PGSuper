1 Define Strands {#ug_dialogs_girder_define_strands}
2 ==============================================
3 Ignore the strand locations defined in the Girder Library and define the strand locations for this girder.
4 
5 Item | Description
6 -----|----------------
7 Strand Grid | Strand rows are defined in this grid. See below for details.
8 [Add] | Add a row to the strand grid
9 [Remove] | Remove the selected row from the strand grid
10 Permanent Strands | Select the permanent strand size and type from the drop down list. If the permanent strands are epoxy coated, check the "Grit impregnated epoxy coating" box. See @ref tg_epoxy_coated_strands in the @ref technical_guide for more information.
11 Temporary Strands | Select the temporary strand size and type from the drop down list. Temporary strands are never epoxy coated.
12 Pjack | For each strand type, the maximum jacking force is shown. Check the box to enter a different jacking force.
13 
14 Strand Grid
15 ------------
16 Strands are defined in horizontal rows. Multiple rows can be defined at the same elevation in the girder section. You must ensure the clear distance between strands is acceptable. The information in the strand row grid is:
17 
18 Item | Description
19 -----|-----------
20 Spacing, S1 | Spacing between the two strands that straddle the vertical centerline of the girder. Enter a value of zero if there is a strand on the vertical centerline.
21 Spacing, S2 | Spacing between strands in a row.
22 Type | Use the drop down list to select the strand type.
23 Left End, Y | Elevation of the strand row at the left end of the girder.
24 Left End, Face | Use the drop down list to select the face of the girder Y is measured from.
25 Left Harp Pt, X | Enter the location of the left harp point from the left end of the girder.
26 Left Harp Pt, Unit | Select the measurement unit for X.
27 Left Harp Pt, Y | Elevation of the strand row at the left harp point of the girder.
28 Left Harp Pt, Face | Use the drop down list to select the face of the girder Y is measured from.
29 Right Harp Pt, X | Enter the location of the right harp point from the left end of the girder.
30 Right Harp Pt, Unit | Select the measurement unit for X.
31 Right Harp Pt, Y | Elevation of the strand row at the right harp point of the girder.
32 Right Harp Pt, Face | Use the drop down list to select the face of the girder Y is measured from.
33 Right End, Y | Elevation of the strand row at the right end of the girder.
34 Right End, Face | Use the drop down list to select the face of the girder Y is measured from.
35 Ext. Strands, Left | Check if the strands are extended at the left end of the girder.
36 Ext. Strands, Right | Check if the strands are extended at the right end of the girder.
37 Debond, Left | Check if the strands are debonded at the left end of the girder. Enter the debond length.
38 Debond, Right | Check if the strands are debonded at the right end of the girder. Enter the debond length.
