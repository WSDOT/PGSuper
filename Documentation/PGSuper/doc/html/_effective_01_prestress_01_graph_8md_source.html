1 Effective Prestress Graph {#ug_effective_prestress_graph}
2 ==============================================
3 The Effective Prestress graph shows how the effective prestress or effective prestress force varies with both time and location for prestressing strands and post-tensioned tendons.
4 
5 The graph controller, found on the left side of the graph window, is used to select the properties to be graphed and other graphing parameters.
6 
7 Graph Controller Item | Description
8 ---------------------|----------------------
9 Span, Girder | Select a span and girder
10 Prestressing Type | Select either pretensioning or post-tensioning graph
11 Permanent/Temporary | If Pretensioning is selected, select either Permanent or Temporary strands
12 Interval | Select the intervals for which to plot the effective prestress. Hold [CTRL] to select multiple, nonsequential, intervals. Hold [SHIFT] and select the first and last intervals in a range.
13 Stress/Force | Select if the effective prestress (fpe) or the effective prestress force (Fpe) is to be plotted
14 Show Grid | Toggles the grid
15 Show Girder | Toggles the girder
