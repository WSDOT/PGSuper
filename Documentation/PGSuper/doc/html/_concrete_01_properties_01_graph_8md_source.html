1 Concrete Properties Graph {#ug_concrete_properties_graph}
2 ==============================================
3 The Concrete Properties graph provides various graphical representation of concrete properties as they vary with time.
4 
5 The graph controller, found on the left side of the graph window, is used to select the properties to be graphed and other graphing parameters.
6 
7 Graph Controller Item | Description
8 ---------------------|----------------------
9 Element | Select the element for which the concrete properties are to be graphed
10 Property | Select the concrete property to be graphed. Creep coefficients are plotted for various times of loading.
11 Horizontal Axis | Select the horizontal axis type. Time plots the concrete properties with respect to the overall analysis timeline. Age plots concrete properties with respect to the age of the concrete. Interval plots concrete properties at discrete time intervals. For the Time and Age options, a linear or logarithmic scale can be used.
12 Show Grid | Toggles the grid.
