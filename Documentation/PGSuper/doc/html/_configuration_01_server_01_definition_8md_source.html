33 ### Local or Network File System Using a .ini File ###
34 This type of server uses a network file system and has the ability to use a .ini file and (*.pgz) files. This has advantages over the normal file system server by allowing multiple configurations at one network location, and versioning of the .pgz configurations. Refer to @ref configuration_server_files for information about how to create .ini and .pgz files.
