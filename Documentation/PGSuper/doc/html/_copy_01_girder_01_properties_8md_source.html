1 Copy Girder Properties {#ug_dialogs_copy_girder_properties}
2 ==============================================
3 It is common practice for precast girder bridge designs to have all girders detailed identically. Copy Girder Properties provides a convenient method for copying the properties of one girder to other girders in the structure.
4 
5 Copy Properties From
6 --------------------
7 Select the source girder whose properties you wish to copy
8 
9 Copy Properties To
10 --------------------
11 Select the target girder(s) to which the properties will be copied. Target girders can be selected with either of two options.
12 
13 Option | Description
14 -------|------------
15 Single Girder, Span, Girderline or All | Using the drop down lists to select the target girders
16 Multiple Girders | Press [Select Girders] to select several target girders
17 
18 Select Types of Properties to be Copied
19 ----------------------------------------
20 Check the box next to the properties to be copied.
21 
22 Property | Description
23 ---------|---------------
24 Copy Concrete Material Properties | When checked, the properties of the girder concrete will be copied.
25 Copy Prestressing | When checked, both pretensioned and postensioned (if present) prestressing strand information (quantities, locations, jacking forces, debonding, etc.) will be copied. For bridges using several types of girders, prestressing information can only be copied between girders that are of the same type.
26 Copy Longitudinal Reinforcement | When checked, the longitudinal reinforcement will be copied. For bridges using several types of girders, longitudinal reinforcement can only be copied between girders that are of the same type.
27 Copy Transverse Reinforcement | When checked, the transverse shear reinforcement will be copied.
28 Copy Handling Locations | When check, the location of the lifting loops and the transportation support points will be copied.
29 Copy Slab Offset ("A" Dimension) | When check, the slab offset is copied.
30 
31 Control Buttons | Description
32 ----------------| -----------
33 [Copy Now] | Press this button to copy the properties.
34 [Close] | Close this window without copying properties
35 [Help] | Opens this help topic.
