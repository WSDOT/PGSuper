Local or Network File System

This type of server is typically used on a computer that is not on a network or for a group of users who want to keep their configration private from the rest of the world.

FTP Server

Local or Network File System Using a .ini File

This type of server uses a network file system and has the ability to use a .ini file and (*.pgz) files. This has advantages over the normal file system server by allowing multiple configurations at one network location, and versioning of the .pgz configurations. Refer to Configuration Servers for information about how to create .ini and .pgz files.
