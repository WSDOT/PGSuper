26 Common Input Parameters
27 ---------------------
28 Station, offset, bearing, and angle are common input parameters used throughout the user interface. These parameters can be input with simple decimal values or in common engineering format.
29 
30 ### Notation ###
31 The following notation is used to describe the common engineering format.
32 
33 <table>
34 <tr><th>Symbol</th><th>Description</th><th>Example</th></tr>
35 <tr><td>|</td><td>Choose between pipe separated items</td><td>L|R<br>Choose "L" or "R"</td></tr>
36 <tr><td>[]</td><td>Items within square brackets are optional</td><td>[+|-]<br>Use "+" or "-" or omit</td></tr>
37 <tr><td>x</td><td>Lowercase letters are digits 0-9</td><td>dd mm ss.ss<br>Angular measure as degree-minute-decimal seconds<br>45 22 34.22
38 </td></tr>
39 </table>

42 ### Stations ###
43 Stations represent a position along an alignment. Stations can be input as a decimal number or as a number of full plus fractional chains.
44 
45 <table>
46 <tr><th>Format</th><th>Examples</th></tr>
47 <tr><td>[+|-]dddd.dd</td><td>233.32<br>-89.92</td></tr>
48 <tr><td>US Units<br>[+|-]dd+dd.dd</td><td>2+33.32<br>-0+89.92</td></tr>
49 <tr><td>SI Units<br>[+|-]dd+ddd.dd</td><td>2+343.32<br>-0+894.92</td></tr>
50 </table>
51 
52 ### Offsets ###
53 Offsets represent a perpendicular distance measured from a reference line. Left and right orientation is based on the observer looking in the positive direction of the reference line. If the reference line is an alignment, left and right are based on the observer looking ahead on station.
54 
55 <table>
56 <tr><th>Format</th><th>Examples</th></tr>
57 <tr><td>[+|-]dddd.dd<br>Values greater than zero are to the right of the reference line.</td><td>2343.32<br>-894.92</td></tr>
58 <tr><td>dddd.dd [L|R]</td><td>2343.32 R<br>894.92 L</td></tr>
59 </table>
60 
61 ### Angles ###
62 Angles represent a departure from a reference line.
63 
64 <table>
65 <tr><th>Format</th><th>Examples</th></tr>
66 <tr><td>[+|-]dddd.dd<br>Clockwise angles are positive.</td><td>45.32<br>-22.92</td></tr>
67 <tr><td>dddd.dd [L|R]<br>Angles to the right are clockwise</td><td>45.32 L<br>22.92 R</td></tr>
68 <tr><td>[+|-]dd [mm [ss.ss]]</td><td>45<br> 45 16<br>45 16 25.32<br>-32<br>-32 56<br>-32 56 14.56</td></tr>
69 <tr><td>dd [mm [ss.ss]] [L|R]</td><td>45 L<br> 45 16 L<br>45 16 25.32 L<br>32 R<br>32 56 R<br>32 56 14.56 R</td></tr>
70 </table>
