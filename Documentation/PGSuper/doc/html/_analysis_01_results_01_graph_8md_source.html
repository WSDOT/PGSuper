1 Analysis Results Graph {#ug_analysis_results_graph}
2 ==============================================
3 The Analysis Results Graph presents analysis results in traditional engineering graph formats including moment, shear, deflection, and stress diagrams.
4 
5 Results can be presented for several different loadings for a specified analysis interval, or they can be shown for a specified loading as it changes over multiple analysis intervals.
6 
7 The graph controller, found on the left side of the graph window, is used to select graphs. The main graph window consists of a title, legend, and the graphical representation of analysis results.
8 
9 Graph Controller Item | Description
10 -----|-------------
11 Plot Mode | Select the plot mode. Select Plot by Interval to plot results for various loadings during a specific interval. Select Plot by Loading to plot results for a specific loading over multiple intervals.
12 Span | Select a span or All Spans
13 Girder | Select a girder, or girder line if All Spans is selected
14 Interval/Loading list | If the plot mode is Plot by Interval, select an interval, otherwise select a loading
15 Results Type | Select an analysis results type to plot such as moment, shear, deflection or stress.
16 Graph Item List | Select one or more items from the graph item list. If the plot mode is Plot by Interval, select loadings, otherwise select intervals. Hold [CTRL] to select multiple, nonsequential, items. Hold [SHIFT] and select the first and last items in a range
17 Show Grid | Toggles the grid
18 Show Girder | Toggles the girder
19 Incremental/Cumulative | Plot incremental results (incremental change in results occuring during the selected interval) or cumulative results (sum of incremental results in this and all preceding intervals).
20 Top Girder | Toggles stress plot for top of girder
21 Bottom Girder | Toggles stress plot for bottom of girder
22 Top Deck | Toggles stress plot for top of deck
23 Bottom Deck | Toggles stress plot for bottom of deck
24 Include Elevation Adjustments | When checked, elevation adjustments are added to computed deflections (PGSplice only)
25 Structural Analysis Method | Select the structural analysis method
26 
27 > NOTE: Analysis results for the Lifting and Hauling Intervals are for plumb girder subjected to self-weight and prestress loading only.
