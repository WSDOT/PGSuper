1 Create a New Project {#tutorial_sidewalks_create_a_new_project}
2 ============
3 New PGSuper projects are created from project templates. The templates contain standard configurations for bridge geometry and components, appurtenances, loading, and design criteria. Since this is a WSDOT WF74G girder bridge, the easiest way to start is with the WF74G template.
4 
5 > TIP: You can customize project templates and their organization. See @ref ug_configurations in the @ref user_guide for more information.
6 
7 Start PGSuper
8 -------------
9 To start PGSuper, select *Start > Programs > Engineering Applications > PGSuper*.
10 
11 Create a New Project
12 ---------------------
13 1. Select *File > New*. This brings up the New Project window.
14 2. Select the WF-Girders Project Type and the WF74G template. ![](Tutorial_Sidewalks_New_Project.png)
15 3. Press [OK].
16 3. This will bring up the Project Properties window where you can enter information about your project. ![](Tutorial_Sidewalks_Project_Properties.png)
17 
18 > NOTE: The Project Properties window may not open when creating a new project. If this is the case, select *File > Project Properties* or *Project > Properties*. Use the check box at the bottom of the window to control whether the window opens when new projects are created.
19 
20 Finally the default configuration for this type of bridge will be displayed in the Bridge Model View.
