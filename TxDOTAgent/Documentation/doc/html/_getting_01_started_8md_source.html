1 Getting Started {#getting_started}
2 ===================
3 TOGA is the software that TxDOT's Bridge Fabrication Branch uses to analyze fabricator optional designs. TOGA is a BridgeLink Application that provides a streamlined user interface for optional design evaluation.
4 
5 [Refer here for TOGA Installation](@ref installation)
6 
7 Creating a new project in TOGA is simple: Open BridgeLink and select *File > New* from the main menu. The following dialog comes up. Then,
8 1. Select the project type "TxDOT Optional Girder Analysis" - Templates in the tree control in the left panel.
9 2. Select the appropriate girder type for your project to open a new TOGA project for that girder, and click [OK].
10 
11 ![](FileNew.gif)
12 
13 > Note: If "TxDOT Optional Girder Analysis" is not an available project type, you probably don't have the TxDOT Extensions (which include TOGA) installed. Re-install BridgeLink with the TxDOT options enabled, or contact technical support if this problem persists.
14 
15 After you have a new file, running TOGA is simple:
16 1. Fill in all of the blank input controls on the [Bridge Input Tab](@ref ui_bridge_input_tab) and [Girder Input Tab ](@ref ui_girder_input_tab)
17 2. Inspect the [Girder View Tab](@ref ui_girder_view_tab) to verify strand input,
18 3. Then visit the [Analysis Report Tab](@ref ui_analysis_report_tab) to review the analysis results.
19 4. Interpret the results and make your decision. Sorry, no help on this one - you're the expert!
20
