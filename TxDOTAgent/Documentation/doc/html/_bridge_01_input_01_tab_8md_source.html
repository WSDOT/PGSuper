1 Bridge Input Tab {#ui_bridge_input_tab}
2 ===================
3 This tab contains bridge-level data related to the project.
4 
5 > Note: All data on this tab must be filled in prior to visiting other tabs.

7 General Information
8 ===================
9 The General Information group contains textual data used to describe the current project in TOGA reports. None of this data affects the analysis.
10 
11 Item | Description
12 -----------|-------------
13 Bridge | Name of the Bridge
14 Bridge ID | TxDOT Bridge Identification Number
15 Job Number | CSJ
16 Engineer | -
17 Company | -
18 Comments | Any additional text to be included in report

21 Design Information
22 ==================
23 This group contains bridge-level design information
24 
25 Item | Description
26 -----------|-------------
27 Span No | Span where the beam is located. (textual information only)
28 Beam No | Beam number. Generally numbered from left to right facing up-station. (textual information only)
29 Beam Type | Type of beam in question. Note that changing beam type will reset the strand data on the Girder Input tab to Standard Strand Fill with zero strands.
30 Span Length | Length of span as measured between bearing centerlines in feet. Note that the overall girder length is determined by the default connection type used by TOGA for the girder in question. This may not reflect the plan or as-fabricated girder length.
31 Beam Spacing | CL to CL spacing of beams in feet.
32 Slab Thickness | Equivalent slab thickness in inches used to compute composite section properties and weight of slab. Slab density is assumed to be 150 pcf.
33 Relative Humidity | In % at bridge location
34 LLDF (Moment) | Live load distribution factor for moment
35 LLDF (Shear) | Live load distribution factor for shear
36

38 Material Properties
39 ===================
40 This group contains bridge-level material properties.
41 
42 Item | Description
43 -----------|-------------
44 Ec, Slab | Modulus of elasticity of slab concrete. Default is 5000.0 KSI.
45 Ec, Beam | Modulus of elasticity of beam concrete for final condition in KSI. Default is 5000.0 KSI.
46 f'c, Slab | Final concrete strength of slab concrete. Default is 4.0 KSI.

49 Design Data
50 ===========
51 This group contains the results from the analysis of the original per-plan beam design.
52 
53 Item | Description
54 -----------|-------------
55 *ft, Design Compressive Stress, Top CL | Beam top fiber stress at centerline, due to total external load. (Service I, Final Stage) in KSI
56 *fb, Design Tensile Stress, Bottom CL | Beam bottom fiber stress at centerline, due to total external load. A negative sign {-} must be entered for tension. (Service III, Final Stage) in KSI
57 Mu, Required Ultimate Moment Capacity | Ultimate moment capacity required in kip-ft
58 
59 > \* Note: Tension is negative
60 
61 Optional Uniform Design Loads {#uniform_design_load}
62 =============================
63 This group contains equivalent uniform design loads to be applied to the girder in the associated stages/load combinations. Default is 0.0 kip/ft. Input is not required
64 
65 Item | Description
66 -----------|-------------
67 W non-comp, DC | Uniform dead load, in addition to weight of beam and slab, applied to non-composite section in the DC load case. (e.g. diaphragms) in kip/ft
68 W comp, DC | Uniform dead load, applied to composite section in the DC load case (e.g., rail) in kip/ft
69 W Overlay | Uniform dead load, applied to composite section in the DW load case after all losses in kip/ft. This is equivalent to a future overlay in PGSuper.
