Getting Started

TOGA is the software that TxDOT's Fabrication Branch uses to analyze fabricator optional designs. TOGA is a PGSuper plug-in application that provides a streamlined user interface for optional design evaluation.

Refer here for TOGA Installation

Creating a new project in TOGA is simple: Open PGSuper and select File|New from the main menu. The following dialog comes up:

Select the appropriate girder type for your project to open a new TOGA project for that girder.

Note: If "TxDOT Optional Girder Analysis" is not an available project type, you probably don't have TOGA installed. Re-install PGSuper with the TxDOT options, or contact technical support if this problem persists.

After you have a new file, running TOGA is simple:

  1. Fill in all of the blank input controls on the Bridge Input Tab and Girder Input Tab

  2. Inspect the Girder View Tab  to verify strand input,

  3. Then visit the Analysis Report Tab to review the analysis results.

  4. Interpret the results and make your decision. Sorry, no help on this one - you're the expert!
