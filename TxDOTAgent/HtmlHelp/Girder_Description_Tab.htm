Strand Fill Information

Strands may be filled using the standard fill sequence, or using a row-by-row definition .

Standard Strand Fill

If the Standard Strand Fill option is checked, strands are filled in the order defined on TxDOT standard precast design plan sheets for the selected girder type (e.g., IBND sheets). This is the simplest way to define strand fill
